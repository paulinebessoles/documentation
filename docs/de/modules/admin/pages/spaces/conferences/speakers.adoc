= Lautsprecher

== Understand the basics

The speakers are the persons that are talking at the conference. They can be linked to meetings, so participants can
see who is going to speak in each item of the program.

image::spaces/conferences/speakers_frontend.png[Conferences speakers frontend]

== Konfiguration

=== Manage speakers

To access the speakers configuration, click in the secondary navigation panel on the "Speakers" item.

image:spaces/conferences/speakers.png[Konferenz-Lautsprecher]

You can search speakers thanks to the search bar.

=== Neuen Lautsprecher erstellen

To create a new speaker, click the "New speaker" button in the top right of the page.

image::spaces/conferences/new_speaker.png[Conferences speakers]

.Neues Konferenz-Lautsprecher-Formular
[cols="20h,~,~"]
|===
|Feld |Typ |Beschreibung

|Teilnehmertyp
|Benötigt
|Wether this speaker is a non-participant or existing participant. Haben sie bereits ein Konto auf der Plattform erstellt?

|Voller Name
|Erforderlich, wenn es nicht teilnimmt.
|Name des Redners.

|Benutzer oder Gruppe
|Erforderlich, wenn es ein existierender Teilnehmer ist.
|Geben Sie mindestens drei Zeichen für die Suche ein.

|Verwandte Besprechungen
|Optional
|The meetings in which the speaker is speaking or facilitating.

|Position
|Benötigt
|Metadaten zum Anzeigen zusätzlicher Informationen über den Lautsprecher.

|Zugehörigkeit
|Benötigt
|Metadaten zum Anzeigen zusätzlicher Informationen über den Lautsprecher.

|Kurze Bio
|Optional
|Metadaten zum Anzeigen zusätzlicher Informationen über den Lautsprecher.

|X handle
|Optional
|Metadaten zum Anzeigen zusätzlicher Informationen über den Lautsprecher.

|Persönliche URL
|Optional
|Metadaten zum Anzeigen zusätzlicher Informationen über den Lautsprecher.

|Avatar
|Optional
|Avatar image for the speaker. Ignored if the speaker is an "Already registered" participant.
|===
