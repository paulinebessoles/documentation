== Konfiguration

To create a Blog component, first go in the xref:admin:spaces.adoc[participatory space] where you want to create it.

image::components/skeleton/component_skeleton.png[Add component button]

Then:

. Click on the "Components" item in the secondary menu
. Click the "Add component" button
. Select "Skeleton"
. Fill the component creation form

image:components/skeleton/component.png[Komponentenform] hinzufügen


.Komponente hinzufügen: Skelett Formular
|===
|Feld |Typ |Beschreibung

|Name
|Benötigt
|Wie lautet der Titel dieser Komponente. Zum Beispiel "Skelette"

|Bestellposition
|Optional
|Which order it has in relation to the other skeletons. Eine niedrigere Zahl bedeutet eine höhere Priorität.
|===


.Komponente hinzufügen: Skelett-Formular - Globale Einstellungen
|===
|Feld |Typ |Beschreibung

|Bereiche aktiviert
|Optional
|Aktivieren, wenn die Bereichsfilterung in dieser Komponente erfolgen soll.

|Bereich
|Optional
|Which xref:admin:settings/scopes.adoc[Scope] does this component belongs to.

|Globaler Umfang
|Optional
|

|Kommentare aktiviert
|Optional
|Ob Sie Kommentare in den Meetings haben möchten.

|Maximale Länge der Kommentare
|Optional
|What is the characters limit that participants have when making comments. Lassen Sie 0 für den Standardwert.

|Aktionsberechtigungen können für jedes Skelett festgelegt werden
|Optional
|Ob Sie Berechtigungen für jedes Skelett festlegen möchten.

|Mitteilung
|Optional
|Eine allgemeine Ankündigung, die auf der Landungsseite der Skelette zu sehen sein wird.
|===

== Neues Skelett Formular

image:components/skeleton/new_skeleton.png[Neue skelett-Form]


.Neues Skelett Formular
|===
|Feld |Typ |Beschreibung

|Titel
|Benötigt
|Wie lautet der Titel dieses Skeletts. Zum Beispiel "Bezirk A"

|Bestellposition
|Optional

|Optional
