== Configuración

To create a Blog component, first go in the xref:admin:spaces.adoc[participatory space] where you want to create it.

image::components/skeleton/component_skeleton.png[Add component button]

Then:

. Click on the "Components" item in the secondary menu
. Click the "Add component" button
. Select "Skeleton"
. Fill the component creation form

image:components/skeleton/component.png[Añadir formulario de componente]


.Añadir componente: formulario de esqueleto
|===
|Campo |Tipo |Descripción

|Nombre
|Requerido
|¿Cuál es el título de este componente. Por ejemplo, "Esqueletones"

|Posición del pedido
|Opcional
|Which order it has in relation to the other skeletons. Tener un número más bajo significa una prioridad mayor.
|===


.Añadir componente: formulario de esqueleto - Configuración global
|===
|Campo |Tipo |Descripción

|Ámbitos habilitados
|Opcional
|Marque si desea tener filtrado de Scopes en este componente.

|Alcance
|Opcional
|Which xref:admin:settings/scopes.adoc[Scope] does this component belongs to.

|Ámbito global
|Opcional
|

|Comentarios habilitados
|Opcional
|Si desea tener comentarios en las reuniones.

|Longitud máxima de los comentarios
|Opcional
|What is the characters limit that participants have when making comments. Deja 0 para el valor predeterminado.

|Se pueden establecer permisos de acciones para cada esqueleto
|Opcional
|Si quieres ser capaz de establecer permisos para cada uno de los esqueletos.

|Anuncio
|Opcional
|Un anuncio general que será visible en la página de inicio general de esqueletos.
|===


.Añadir componente: Formulario de esqueletos - Ajustes de paso
|===
|Campo |Tipo |Descripción

|Comentarios bloqueados
|Opcional
|Si desea habilitar los comentarios para esta fase.

|Anuncio
|Opcional
|Un anuncio general que será visible en la página general de aterrizaje del esqueleto .
|===
