= Socios

== Understand the basics

Los socios son los patrocinadores o partidarios de una conferencia.
If you add partners in the administration, then they are listed on the Conference landing page.

image:spaces/conferences/partners_frontend.png[Conferences partners frontend]

== Configuración

=== Manage partners

To access the partners configuration, click in the secondary navigation panel on the "Partners" item.

=== Crear un nuevo socio

To create a new partner, click the "New partner" button in the top right of the page.

image::spaces/conferences/new_partner.png[Conferences partners form]

.Nuevo formulario de socio
[cols="20h,10h,~"]
|===
|Campo |Tipo |Descripción

|Nombre
|Requerido
|The name of the partner.

|Tipo de socio
|Requerido
|Choose one of these options: Main promoter or Collaborator.

|Posición del pedido
|Opcional
|Which order it has in relation to the other partners. Tener un número más bajo significa una prioridad mayor. For instance, if you have two partners, "Partner a" with weight "10" and "Partner b" with weight "0" then "Partner b" is first when ordering.

|Enlace
|Opcional
|Link to the partner website, bio, social media, etc.

|Logo
|Requerido
|Guía para la imagen: Preferiblemente una imagen de paisaje que no tiene ningún texto. El servicio cultiva la imagen. Maximum file size: 5 MB
|===
