= Adjuntos

== Understand the basics

Los archivos adjuntos permiten a un administrador subir archivos estáticos a un espacio.

Cuando se lanza un espacio, es importante proporcionar a los ciudadanos toda la información posible sobre ellos para que puedan participar con conocimiento.
That is why it is helpful to add files, documents, images with relevant information.

These documents can be grouped in *Folders*. Por ejemplo, para un proceso participativo de remodelación de un cuadrado, se podrían crear carpetas con información sobre posibles usos, con planes, con información económica, etc.

They can be found on the space xref:admin:spaces/processes/landing_page.adoc[landing page] when the content block is enabled.

image:spaces/attachments.png[Ver los archivos adjuntos en la interfaz pública]

=== How to find it?

To create and manage attachments, click the *Attachments* item in the left navigation bar of any participatory space.

image:spaces/attachments_admin.png[View the attachments in the administration interface]

There, you are able to manage the attachments and folders, configure them, add new ones, etc.

== Configuración

=== Crear un nuevo archivo adjunto

To create a new attachment, click the "New attachment" button.

image:spaces/attachments_new_form.png[Crear un nuevo formulario adjunto]

.Crear archivo adjunto
[cols="20h,10h,~"]
|===
|Campo |Tipo |Descripción

|Adjunto o nombre de imagen
|Requerido
|¿Cuál es el título de este archivo adjunto?

|Posición del pedido
|Opcional
|Which order it has in relation to the other attachments. Tener un número más bajo significa una prioridad mayor. For instance, if you have two categories, "Attachment a" with weight "10" and "Attachment b" with weight "0" then "Attachment b" will be first when ordering.

|Descripción
|Requerido
|A long description explaining what this attachment is about.

|Carpeta
|Opcional
|A qué carpeta pertenece este archivo adjunto.

|Fichero
|Requerido
|Subir el archivo. Guía para el archivo: Debe ser una imagen o un documento. Maximum file size: 10 MB. Allowed
file extensions depends in your server configuration.
|===

=== Crear una nueva carpeta

image:spaces/attachments_folders_admin.png[View the folders in the administration interface]

To create and manage attachments folders, click the *Folders* tab.

image:spaces/attachments_new_folder_form.png[Crear una nueva carpeta para el formulario de adjuntos]

.Crear carpeta
[cols="20h,10h,~"]
|===
|Campo |Tipo |Descripción

|Nombre
|Requerido
|Nombre de la carpeta

|Posición del pedido
|Opcional
|Which order it has in relation to the other folders. Tener un número más bajo significa una prioridad mayor. For instance, if you have two folders, "Folder a" with weight "10" and "Folder b" with weight "0" then "Folder b" will be first when ordering.

|Descripción
|Requerido
|Una descripción larga que explica de qué trata esta carpeta.
|===

=== Manage attachments

You can find all the attachments on the list in the administration panel, with:

* *Title*: title of the attachment, given when creating or editing it.
* *Folder*: folder where the attachment is stored.
* *Type*: extension type of the attachment.
* *Size*: size of the attachment in bytes.

You can also access the folders list by clicking the "Folders" tab on top of the attachments list.
