= Documentación

Para contribuir a la documentación (por ejemplo, esta misma página que estás leyendo), puede hacerlo a través de dos maneras: editar directamente en GitHub o editar localmente.

== Edit on GitHub

Esta es la forma recomendada si desea hacer una corrección rápida (por ejemplo, un error tipográfico o alguna aclaración en una sola página). Para hacer esto tendrás que hacer clic en el enlace "Editar esta página" en la parte superior de cada página.

image::documentation-edit.png[Edit this page link]

Abrirá la página para editar en la interfaz de GitHub y después de que hayas terminado te guiará a hacer una Pull Request para que los mantenedores puedan revisar tu contribución.

== Editar localmente

Si desea proponer muchos cambios en la documentación, o si desea revisar sus cambios localmente antes de enviarlos porque quiere revisar cosas como el formato, puede instalar el entorno para tener el sitio de documentación localmente.

Entonces necesitará bifurcar el repositorio, hacer una rama y proponer sus cambios como una Pull Request para que los mantenedores puedan revisarlo. Como ves esto puede ser un problema si quieres hacer pequeños cambios, así que la forma preferida sería hacerlo inicialmente con el enlace "Editar esta página" mehod.

Lea las instrucciones completas sobre cómo instalar este sitio web de documentación localmente en https://github.com/decidim/documentation/[Repositorio de documentación de Decidim GitHub].
