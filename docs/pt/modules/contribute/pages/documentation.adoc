= Documentação

Para contribuir com a documentação (por exemplo, esta mesma página que está lendo), você pode fazer isso de duas maneiras: editar no GitHub diretamente ou editar localmente.

== Edit on GitHub

Esta é a forma recomendada se você só quiser fazer uma correção rápida (por exemplo, um erro de digitação ou algum esclarecimento em uma única página). Para fazer isso, você precisará clicar no link "Editar esta página" no topo de cada página.

image::documentation-edit.png[Edit this page link]

Ele abrirá a página para edição na interface do GitHub e, depois de terminar, o guia para criar um Pull Request para que os mantenedores possam revisar a sua contribuição.

== Editar localmente

Se você quiser propor muitas alterações na documentação, ou se você quiser revisar suas alterações localmente antes de enviá-las porque deseja revisar coisas como formatação, Você pode instalar o ambiente para ter o site de documentação localmente.

Em seguida, você precisará criar um fork do repositório, fazer um branch e propor as alterações como um Pull Request para que os mantenedores possam revisá-lo. Como você vê, isto pode ser uma complicação se você quiser fazer pequenas alterações, assim a forma preferida seria fazer isso inicialmente com o link "Editar esta página".

Leia as instruções completas sobre como instalar localmente esta documentação no site https://github.com/decidim/documentation/[Repositório do GitHub da Documentação Decidim.
