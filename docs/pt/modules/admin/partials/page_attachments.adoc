= Anexos

== Understand the basics

Anexos permitem que um administrador faça upload de arquivos estáticos para um espaço.

Quando um espaço é lançado, é importante fornecer aos cidadãos toda a informação possível sobre eles, para que possam participar com conhecimentos.
That is why it is helpful to add files, documents, images with relevant information.

These documents can be grouped in *Folders*. Por exemplo, para um processo participativo de remodelação de um quadrado, poderiam ser criadas pastas com informação sobre possíveis usos, com planos, com informações económicas, etc.

They can be found on the space xref:admin:spaces/processes/landing_page.adoc[landing page] when the content block is enabled.

imagem:spaces/anexos.png[Veja os anexos na interface pública]

=== How to find it?

To create and manage attachments, click the *Attachments* item in the left navigation bar of any participatory space.

image:spaces/attachments_admin.png[View the attachments in the administration interface]

There, you are able to manage the attachments and folders, configure them, add new ones, etc.

== Configuração

=== Criar um novo anexo

To create a new attachment, click the "New attachment" button.

imagem:spaces/attachments_new_form.png[Crie um novo formulário de anexo]

.Criar anexo
[cols="20h,10h,~"]
|===
|Campo |tipo |Descrição:

|Nome de imagem ou anexo
|Obrigatório
|Qual é o título para este anexo.

|Posição do pedido
|Opcional
|Which order it has in relation to the other attachments. Ter um número menor significa uma prioridade mais alta. For instance, if you have two categories, "Attachment a" with weight "10" and "Attachment b" with weight "0" then "Attachment b" will be first when ordering.

|Descrição:
|Obrigatório
|A long description explaining what this attachment is about.

|pasta
|Opcional
|Qual pasta pertence a este anexo.

|Arquivo
|Obrigatório
|Enviar o arquivo. Guia para o arquivo: Tem que ser uma imagem ou um documento. Maximum file size: 10 MB. Extensões
permitidas dependem da configuração do seu servidor.
|===

=== Criar nova pasta

image:spaces/attachments_folders_admin.png[View the folders in the administration interface]

|===

|Posição do pedido

|Descrição:
|Obrigatório
|Uma longa descrição explicando sobre o que é essa pasta.
|===

=== Manage attachments

You can find all the attachments on the list in the administration panel, with:

* *Title*: title of the attachment, given when creating or editing it.
* *Folder*: folder where the attachment is stored.
* *Type*: extension type of the attachment.
* *Size*: size of the attachment in bytes.

You can also access the folders list by clicking the "Folders" tab on top of the attachments list.
