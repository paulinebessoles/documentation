= Parceiros

== Understand the basics

Os parceiros são os patrocinadores ou apoiadores de uma conferência.
If you add partners in the administration, then they are listed on the Conference landing page.

image:spaces/conferences/partners_frontend.png[Conferences partners frontend]

== Configuração

=== Manage partners

To access the partners configuration, click in the secondary navigation panel on the "Partners" item.

=== Criar um novo parceiro

To create a new partner, click the "New partner" button in the top right of the page.

image::spaces/conferences/new_partner.png[Conferences partners form]

.Novo formulário de parceiro
[cols="20h,10h,~"]
|===
|Campo |tipo |Descrição:

|Nome:
|Obrigatório
|The name of the partner.

|Tipo de parceiro
|Obrigatório
|Choose one of these options: Main promoter or Collaborator.

|Posição do pedido
|Opcional
|Which order it has in relation to the other partners. Ter um número menor significa uma prioridade mais alta. For instance, if you have two partners, "Partner a" with weight "10" and "Partner b" with weight "0" then "Partner b" is first when ordering.

|Vincular
|Opcional
|Link to the partner website, bio, social media, etc.

|Logotipo
|Obrigatório
|Orientação para imagem: Preferencialmente, uma imagem de paisagem que não possui qualquer texto. O serviço cultiva a imagem. Maximum file size: 5 MB
|===
