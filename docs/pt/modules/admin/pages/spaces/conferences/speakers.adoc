= Alto-falantes

== Understand the basics

The speakers are the persons that are talking at the conference. They can be linked to meetings, so participants can
see who is going to speak in each item of the program.

image::spaces/conferences/speakers_frontend.png[Conferences speakers frontend]

== Configuração

=== Manage speakers

To access the speakers configuration, click in the secondary navigation panel on the "Speakers" item.

imagem:espaçaces/conferências/palestrantes.png[Palestrantes de Conferências]

You can search speakers thanks to the search bar.

=== Criar um novo pregador

To create a new speaker, click the "New speaker" button in the top right of the page.

image::spaces/conferences/new_speaker.png[Conferences speakers]

.Novo formulário de palestrante de conferência
[cols="20h,~,~"]
|===
|Campo |tipo |Descrição:

|Tipo de usuário
|Obrigatório
|Wether this speaker is a non-participant or existing participant. Já têm uma conta criada na plataforma?

|Nome completo
|Necessário se não for participante.
|Nome do palestrante.

|Usuário ou grupo
|Necessário se for um participante existente.
|Digite pelo menos três caracteres para pesquisar.

|Reuniões relacionadas
|Opcional
|The meetings in which the speaker is speaking or facilitating.

|Posição
|Obrigatório
|Metadados para mostrar informações adicionais sobre o palestrante.

|Afiliação
|Obrigatório
|Metadados para mostrar informações adicionais sobre o palestrante.

|Pequena biografia
|Opcional
|Metadados para mostrar informações adicionais sobre o palestrante.

|X handle
|Opcional
|Metadados para mostrar informações adicionais sobre o palestrante.

|URL pessoal
|Opcional
|Metadados para mostrar informações adicionais sobre o palestrante.

|Avatar
|Opcional
|Avatar image for the speaker. Ignored if the speaker is an "Already registered" participant.
|===
