= Compromissos

== Understand the basics

O recurso de agenda permite que um administrador defina o calendário para uma reunião. O recurso de agenda permite que um administrador defina o calendário para uma reunião. It's possible to define agenda items and
sub-items, with every one of the topics to be discussed in the meeting, along with a description and the duration
of every item and sub-item.

== Configuração

=== Novo formulário de agenda

To define an agenda for a meeting, click the image:icons/action_agenda.png[Agenda icon] Agenda icon.

image:components/meetings/agenda/new_agenda_form.png[Novo formulário agenda no backend]

.Novo formulário de agenda
[cols="20h,10h,~"]
|===
|Campo |tipo |Descrição:

|Título
|Obrigatório
|Título para a agenda.

|Visível
|Opcional
|Check if you want the agenda to be visible to participants.

|Itens da agenda
|Obrigatório
|Lista de itens da agenda.
|===

É possível criar qualquer número de itens da Agenda e subitens. Você pode subir ou descer a hierarquia dos itens da agenda.
Também é possível remover e editar os itens da agenda.

.Formulário item da agenda
[cols="20h,10h,~"]
|===
|Campo |tipo |Descrição:

|Título
|Obrigatório
|Título do item da agenda

|Duração
|Obrigatório
|How many minutes the agenda item last.

|Descrição:
|Opcional
|Title for the agenda item.

|Subitens da agenda
|Opcional
|Lista de sub-itens da agenda.
|===

A única limitação que os pontos da ordem do dia e os sub-pontos têm é o facto de terem de respeitar a duração do progenitor. For instance,
if there's an item that lasts 60 minutes, you can't create a sub-item that lasts 90 minutes.
