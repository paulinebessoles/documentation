= Blogue

== Understand the basics

The Blog component allows administrators to post news about a participatory space, which participants can read to get updates.

image::components/blog/example01.png[Example of blog posts list]

All blog posts are available on the post list, and participants can click each blog post to learn more about what's new.

image::components/blog/example02.png[Example of a blog post]

== Configuração

=== Create a blog component

To create a Blog component, first go in the xref:admin:spaces.adoc[participatory space] where you want to create it.

image::components/blog/component_blog.png[Add Blog component button]

Then:

. Click on the "Components" item in the secondary menu
. Click the "Add component" button
. Select "Blog"
. Fill the component creation form

You can still update this form and the configuration after creating the component, by clicking the
image:icons/action_configure.png[Configure icon] icon on the components list.

image::components/blog/component.png[Add component form]

.Adicionar componente: Formulário Blog
[cols="20h,10h,~"]
|===
|Campo |tipo |Descrição:

|Nome:
|Obrigatório
|The title of this component. For instance, "Blogs" or "Latest news"

|Posição do pedido
|Opcional
|Which order it has in relation to the other components. Ter um número menor significa uma prioridade mais alta.
|===

.Blog form - Global settings
[cols="20h,10h,~"]
|===
|Campo |tipo |Descrição:

|Anúncio
|Opcional
|A general announcement visible on the Blog component landing page.

|Comentários ativados
|Opcional
|If you want participants to be able to comment on the blog posts.

|Comprimento máximo de comentários
|Opcional
|The characters limit that participants have when making comments. Deixe 0 para o valor padrão.
|===

.Step settings
[cols="20h,10h,~"]
|===
|Campo |tipo |Descrição:

|Anúncio
|Opcional
|A general announcement that will be visible on the general blog landing page.

|Comentários bloqueados
|Opcional
|Se você quer habilitar a criação de comentários para esta fase.

|Endossos ativados
|Opcional
|Se você deseja habilitar endossos para esta fase.

|Endossos bloqueados
|Opcional
|Whether you want to enable adding new endorsements for this phase.
|===

=== Gerenciar publicações

To start creating and managing posts, click on the component title in the navigation bar or on the component list, or click on the
image:icons/action_edit.png[Manage component icon] icon on the components list.

image:components/blog/manage_posts.png[Gerenciar tabela de postagens]

=== Create a post

To create a post, click the "New post" button on the top right of the page.

image:components/blog/new_post.png[Novo formulário de postagem]

.Novo formulário de publicação
[cols="20h,10h,~"]
|===
|Campo |tipo |Descrição:

|Author
|Obrigatório
|Participant account or user group authoring the post.

|Título
|Obrigatório
|Qual é o título deste post.

|Conteúdo
|Obrigatório
|Qual é a descrição deste post.

|Publish time
|Opcional
|Publication date for the post. If in the future, the post is not published until this date.
|===
