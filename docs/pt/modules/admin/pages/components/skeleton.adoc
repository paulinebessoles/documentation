== Configuração

To create a Blog component, first go in the xref:admin:spaces.adoc[participatory space] where you want to create it.

image::components/skeleton/component_skeleton.png[Add component button]

Then:

. Click on the "Components" item in the secondary menu
. Click the "Add component" button
. Select "Skeleton"
. Fill the component creation form

image:components/skeleton/component.png[Adicionar formulário de componente]


.Adicionar componente: Formulário Esqueleto
|===
|Campo |tipo |Descrição:

|Nome:
|Obrigatório
|Qual é o título deste componente. Por exemplo, "Esqueletos"

|Posição do pedido
|Opcional
|Which order it has in relation to the other skeletons. Ter um número menor significa uma prioridade mais alta.
|===


.Adicionar componente: Formulário Esqueleto - Configurações globais
|===
|Campo |tipo |Descrição:

|Âmbitos ativados
|Opcional
|Verifique se você quer ter filtragem de escopos neste componente.

|Escopo
|Opcional
|Which xref:admin:settings/scopes.adoc[Scope] does this component belongs to.

|Âmbito global
|Opcional
|

|Comentários ativados
|Opcional
|Se você deseja ter comentários nas reuniões.

|Comprimento máximo de comentários
|Opcional
|What is the characters limit that participants have when making comments. Deixe 0 para o valor padrão.

|Permissões de ações podem ser definidas para cada esqueleto
|Opcional
|Se você quer ser capaz de definir permissões para cada um dos esqueletos.

|Anúncio
|Opcional
|Um anúncio geral que será visível na página inicial dos esqueletos gerais.
|===


.Adicionar componente: Formulário de esqueletos - Configurações de passos
|===
|Campo |tipo |Descrição:

|Comentários bloqueados
|Opcional
|Se você quer habilitar comentários para esta fase.

|Anúncio
|Opcional
|Um anúncio geral que será visível na página inicial do esqueleto geral.
|===
