= Specific configurations

== Understand the basics

The Proposals component has been built with different layouts, in order to address different use cases.
Those specific configurations are more rarely used, and require a good knowledge of the software.

== Configuração

All three features can be enabled in the Global settings of the proposal component configuration form.
