= Fichiers joints

== En bref

Les pièces jointes permettent à unx administrateur·ice de télécharger des fichiers statiques dans un espace.

Quand un espace est lancé, il est important de fournir aux citoyen·ne·x·s autant d'informations que possible à leur sujet afin qu'iles puissent participer en connaissance de cause.
C'est pourquoi il est utile d'ajouter des fichiers, des documents, des images avec des informations pertinentes.

Ces documents peuvent être regroupés dans *Dossiers*. Par exemple, pour une concertation de remodelage d'une place publique, des dossiers peuvent être créés avec des informations sur les utilisations possibles, avec des plans, avec des informations économiques, etc.

Les documents se trouvent dans l'espace xref:admin:spaces/processes/landing_page.adoc[page d'accueil] lorsque le bloc de contenu est activé.

image:espaces/attachments.png[Voir les pièces jointes dans l'interface publique]

=== How to find it?

Pour créer et gérer les pièces jointes, cliquez sur *Pièces jointes* dans la barre de navigation de gauche de tout espace participatif.

image:espaces/attachments_admin.png[Voir les pièces jointes dans l'interface d'administration]

Là, vous pouvez gérer les pièces jointes et les dossiers, les configurer, en ajouter de nouvelles, etc.

== Configuration

=== Créer une nouvelle pièce jointe

Pour créer une nouvelle pièce jointe, cliquez sur "Nouvelle pièce jointe".

image:espaces/attachments_new_form.png[Créer un nouveau formulaire de pièces jointes]

.Créer une pièce jointe
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Nom de la pièce jointe ou de l'image
|Obligatoire
|Quel est le titre de cette pièce jointe.

|Rang d'affichage
|Facultatif
|Quel ordre il a en relation avec les autres pièces jointes. Avoir un nombre inférieur signifie un rang d'affichage plus élevé. Par exemple, si vous avez deux catégories, « Pièce jointe a » de poids « 10 » et « Pièce jointe b » de poids « 0 » puis « Pièce jointe b » apparaitra en premier dans le listage.

|Description
|Obligatoire
|Une longue description expliquant en quoi consiste cette pièce jointe.

|Répertoire
|Facultatif
|À quel dossier appartient cette pièce jointe.

|Fichier
|Obligatoire

=== Créer un nouveau dossier

image:espaces/attachments_folders_admin.png[Voir les dossiers dans l'interface d'administration]

Pour créer et gérer les dossiers de pièces jointes, cliquez sur l'onglet *Dossiers*.

image:espaces/attachments_new_folder_form.png[Formulaire de nouveau dossier de pièces jointes]

.Créer un dossier
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Nom
|Obligatoire
|Nom du dossier

|Rang d'affichage
|Facultatif
|Quel ordre il a en relation avec les autres dossiers. Avoir un nombre inférieur signifie un rang d'affichage plus élevé. Par exemple, si vous avez deux dossiers, "Dossier a" de poids "10" et "Dossier b" de poids "0" puis "Dossier b" apparaitra en premier dans le listage.

|Description
|Obligatoire
|Une longue description expliquant le dossier.
|===

=== Gérer les pièces jointes

Vous pouvez trouver toutes les pièces jointes dans la liste dans le panneau d'administration, avec :

* *Titre* : titre de la pièce jointe, donné lors de la création ou de l'édition.
* *Dossier* : dossier où la pièce jointe est stockée.
* *Type* : type d'extension de la pièce jointe.
* *Taille* : taille de la pièce jointe en octets.

Vous pouvez également accéder à la liste des dossiers en cliquant sur l'onglet "Dossiers" en haut de la liste des pièces jointes.
