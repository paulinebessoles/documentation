= Administrateur·ices d'espace

== En bref

Vous pouvez donner des permissions spécifiques aux utilisateur·ices dans les différents espaces, et ne peuvent donc avoir accès qu'à certaines actions d'administration.
Ces rôles sont administrateur·ices, modérateur·ices, collaborateur·ices et des évaluateur·ices.

* *Administrateur·ices* : peuvent administrer l'espace et modifier les paramètres de l'espace.
* *Collaborateur·ices* : peuvent voir l'espace et ses fonctionnalités avant leur publication et voir les paramètres de l'espace.
Iels ne peuvent rien modifier.
* *Modérateur·ices* : peuvent voir le contenu signalé de l'espace et le masquer ou le dévoiler.
* *Évaluateur·ices* : peuvent évaluer les propositions, leur donner des réponses et mettre à jour leurs progrès.

=== How to find it?

Pour configurer les administrateur·ices d'espace, cliquez sur l'élément "Administrateur·ices de concertation" ou "Administrateur·ices de l'assemblée" dans l'espace correspondant.

image::spaces/admins_list.png[Administrators of space]

== Configuration

=== Gérer les administrateur·ices

Vous pouvez trouver touxtes les administrateur·ices dans la liste dans le panneau d'administration, avec :

* *Nom* : nom du compte administrateur·ice.
* *E-mail* : e-mail du compte administrateur·ice.
* *Date de dernière connexion* : dernière fois que l'administrateur·ice s'est connecté·e·x.
* *Invitation acceptée le* : date à laquelle l'administrateur·ice a accepté l'invitation en cliquant sur le lien dans l'e-mail d'invitation.
* *Rôle* : le rôle de l'administrateur·ice.

=== Rôles

==== Administrateur·ices

Unx administrateur·ice de l'espace peut administrer l'espace et modifier les paramètres de l'espace. Il peut également inviter de nouveaulx administrateur·ices de l'espace.
Lors de la gestion de l'espace, iel ne voit que les vues et les onglets suivants.

image:espaces/admins_role_admin01.png[Vue des admins de processus 01]

image:espaces/admins_role_admin02.png[Vue des admins de processus 02]

==== Collaborateur·ices

Unx collaborateur·ice de l'espace peut prévisualiser un espace avant sa publication.

image:espaces/admins_role_collaborator.png[Vue des collaborateur·ixces]

==== Modérateur·ices

Unx modérateur·ice d'espace peut examiner le contenu signalé de l'espace et les masquer ou les dévoiler. Iel peut aussi
accéder au contenu signalé dans le panneau de modération globale, mais ne verra pas le contenu signalé depuis d'autres espaces.

image:espaces/admins_role_moderator.png[Vue des modérateur·ixces]

==== Évaluateur·ices

Unx évaluateur·ice d'espace peut examiner les propositions de l'espace qui lui ont été attribuées par:

* modifier le statut des propositions en les acceptant ou en les rejetant.
* donner une réponse aux propositions.
* ajouter une valeur monétaire aux propositions, afin qu'elles puissent être votées avec la fonctionnalité budgétaire.

image:espaces/admins_role_valuator.png[Vue des évaluateur·ixces de processus]
