= Administrateurs de l'espace

== Understand the basics

Vous pouvez donner des permissions spécifiques aux utilisateurs dans les différents espaces, donc ils ne peuvent avoir accès qu'à certaines actions d'administration.
Ces rôles sont des administrateurs, des modérateurs, des collaborateurs et des évaluateurs.

* *Administrateurs* : ils peuvent administrer l'espace et modifier les paramètres de l'espace.
* *Collaborateurs* : ils peuvent voir l'espace et ses composants avant leur publication et voir les paramètres de l'espace.
Ils ne peuvent rien modifier.
* *Modérateurs* : ils peuvent voir le contenu signalé de l'espace et le masquer ou le dévoiler.
* *Valuators* : ils peuvent évaluer les propositions, leur donner des réponses et mettre à jour leurs statuts.

=== How to find it?

Pour configurer les administrateurs de l'espace, cliquez sur l'élément "Administrateurs du processus" ou "Administrateurs de l'assemblée" dans l'espace correspondant.

== Configuration

=== Gérer les administrateurs

Vous pouvez trouver tous les administrateurs dans la liste dans le panneau d'administration, avec:

* *Nom*: nom du compte administrateur.
* *E-mail* : e-mail du compte administrateur.
* *Date de dernière connexion*: dernière fois que l'administrateur a été connecté.
* *Invitation acceptée à* : date à laquelle l'administrateur a accepté l'invitation en cliquant sur le lien dans l'e-mail d'invitation.
* *Role*: administrator role.

Cliquez sur le bouton *Nouvel espace admin* (selon le type d'espace dans lequel vous êtes).

=== Administrateurs

Un administrateur de l'espace peut administrer l'espace et modifier les paramètres de l'espace. Ils peuvent également inviter de nouveaux administrateurs de l'espace.
Lors de la gestion de l'espace, ils ne verraient que les vues et les onglets suivants.

image:espaces/admins_role_admin01.png[Vue des administrateurs de processus 01]

image:espaces/admins_role_admin02.png[Vue des administrateurs de processus 02]

=== Collaborateurs

Un collaborateur de l'espace peut prévisualiser un espace avant sa publication.

image:espaces/admins_role_collaborator.png[Vue des collaborateurs]

=== Modérateurs

Un modérateur d'espace peut examiner le contenu signalé de l'espace et les masquer ou les dévoiler. Ils peuvent aussi
accéder au contenu signalé dans le panneau de modération globale, mais ne voir pas le contenu rapporté depuis d'autres espaces.

image:espaces/admins_role_moderator.png[Vue des modérateurs]

=== Évaluateurs

Un évaluateur d'espace peut examiner les propositions de l'espace qui leur ont été attribuées par:

* modifier le statut des propositions en les acceptant ou en les rejetant.
* donner une réponse aux propositions.
* ajouter une valeur monétaire aux propositions, afin qu'elles puissent être votées avec la composante budgétaire.

image:espaces/admins_role_valuator.png[Vue des évaluateurs de processus]
