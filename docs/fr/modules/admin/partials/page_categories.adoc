= Catégories

== En bref

Les catégories (et sous-catégories) sont utilisées pour classer les ressources à l'intérieur d'un espace.

Unx administrateur·ice peut définir les catégories, peut être affecté lors de la création de ressources, et les participant·e·x·s peuvent ensuite filtrer les
différentes ressources par ces catégories. Par exemple, elles peuvent être utilisées dans des propositions, des réunions et des débats.

. Les administrateur·ices définissent les catégories dans un espace.
. Les participant·e·x·s peuvent créer des ressources et leur donner une catégorie.
. Les participant·e·x·s peuvent ensuite filtrer par catégories.

image::spaces/categories_sidebar.png[Filter categories in proposals]

NOTE: Une fois une catégorie utilisée, elle ne peut pas être supprimée, car il y a des ressources qui y sont liées.

Decidim a d'autres moyens de classer les ressources, par exemple xref:admin:settings/scopes.adoc[Secteurs] et
xref:admin:features/social_features/hashtags.adoc[Hashtags].
La principale différence est que les Catégories peuvent être définies dans un espace donné, tandis que les Secteurs sont généraux pour toute la plateforme et les Hashtags
peuvent être définis par les participant·e·x·s.

Les catégories peuvent être imbriquées, ce qui signifie qu'une catégorie peut avoir des sous-catégories.

=== How to find it?

Pour configurer les catégories, cliquer sur Catégories dans un espace participatif.

image::spaces/categories.png[View categories in a space administration page]

== Configuration

=== Créer une nouvelle catégorie

Cliquez sur le bouton "Nouvelle catégorie" en haut à droite.

image:espaces/categories_new_form.png[Créer un nouveau formulaire de catégorie]

.Formulaire Créer une nouvelle catégorie
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Nom
|Obligatoire
|Quel est le nom de cette catégorie.

|Rang d'affichage
|Facultatif
|Quel ordre il a en relation avec les autres catégories. Avoir un nombre inférieur signifie un rang d'affichage plus élevé. Par exemple, si vous avez deux catégories, "Catégorie a" de poids "10" et "Catégorie b" de poids "0", alors "Catégorie b" apparaitra en premier dans le listage.

|Parent
|Facultatif
|Quelle est la catégorie parente?
|===

=== Gérer les catégories

Vous pouvez trouver toutes les catégories dans la liste dans le panneau d'administration, avec:

* *Nom*: nom de la catégorie, donné lors de la création ou de la modification par l'administrateur·ice.
