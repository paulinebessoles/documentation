= Participant·e·x·s privé·e·x·s

== En bref

Dans certains cas, il est possible de restreindre la participation à un groupe spécifique de participant·e·x·s.
Par exemple, un espace dédié aux membres d'une commission au sein d'une grande organisation.
Pour ce faire, il est possible d'utiliser la fonctionnalité Participant·e·x·s privé·e·x·s.

NOTE: La liste des Participant·e·x·s privé·e·x·s peut être consultée uniquement dans le panneau d'administration et n'est pas affichée publiquement.

=== How to find it?

NOTE: Les Participant·e·x·s privé·e·x·s sont atteignables uniquement dans des espaces privés. Pour les activer, s'assurer d'avoir coché la case "Espace privé" lors de la création ou de la modification de votre concertation ou de votre assemblée.

Pour créer et gérer des participant·e·x·s privé·e·x·s, cliquer sur "Participant·e·x·s privé·e·x·s" dans la barre de navigation à gauche de l'espace participatif.

image::spaces/private_participants.png[Private participants list]

Lé, il est possible de gérer les participant·e·x·s privé·e·x·s, les importer dans un batch, ajouter des participant·e·x·s manuellement, etc.

=== Importer via CSV

Cliquer sur "Importer via CSV".

image::spaces/private_participants_csv_import.png[Import private participants via CSV]

Ce formulaire permet de supprimer l'accès de touxtes les participant·e·x·s privé·e·x·s en une fois, et à regrouper l'importation via CSV.

Le CSV doit suivre ces étapes de formattage :

* Le fichier doit contenir deux colonnes : les emails et les noms des participant·e·x·s à exporter.
* Le fichier ne doit pas avoir d'en-têtes. Ceci signifie qu'il n'est pas nécessaire d'ajouter une ligne qui indique "email, nom"
* Il est important d'éviter des symboles invalides dans le nom, tels que `<>?%&^*#@()[]=+:;"{}\|`.
* Le séparateur entre colonnes doit être une virgule (",").

=== Invitation

NOTE: Si le mail est déjà lié à un compte de l'instance, lae participant·e·x ne recevra par de mail d'invitation, mais sera ajouté·e·x à l'espace participatif.

Lorsqu'unx participant·e·x est ajouté·e·x à une liste de participant·e·x·s privé·e·x·s, mais n'a pas encore de compte, il leur parvient un mail contenant un lien pour accepter l'invitation :

image::spaces/private_participants_email_invite.png[Email invitation of a private participant]

Après avoir cliqué sur le lien "Accepter invitation" dans le mail, le participant·e·x est redirigé·e·x sur l'instance pour compléter la création de son compte en ajoutant son surnom, mot de passe et en acceptant les Conditions.

image::spaces/private_participants_user_registration_form.png[User registration form of a private participant]

=== Gérer les Participant·e·x·s privé·e·x·s

Les participant·e·x·s privé·e·x·s se trouvent dans la liste du panneau d'administration, avec :

* *Nom* : Nom de lae participant·e·x privé·e·x. Si un compte existe déjà, le nom est automatiquement remplacé par le nom d'utilisateur·ice du compte.
* *Email* : Email de lae participant·e·x.
* *Invitation envoyé le* : Le jour et l'heure à laquelle l'invitation a été envoyée (si lae participant·e·x n'a pas déjà un compte).
* *Invitation acceptée le* : Le jour et l'heure à laquelle le mail d'invitation a été accepté par lae participant·e·x.

Il est possible de trier la liste en cliquant sur l'en-tête des colonnes, la filtrer et chercher les participant·e·x·s par leur nom et adresse mail.
