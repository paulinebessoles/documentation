= Page d'accueil

== En bref

Chaque concertation et assemblée détient une page d'accueil, avec différents blocs de contenu qui peuvent être ajoutés, supprimé, activé, ordonné et configuré,
comme dans xref:admin:settings/homepage[Page d'accueil].

Pour organiser une page d'accueil, atteindre l'espace et cliquer sur l'élément de la page d'accueil dans la barre de navigation de gauche.
Vous êtes redirigé·e·x vers le panneau de configuration de la page d'accueil.

. Ajouter différents blocs de contenu en cliquant sur le bouton « Ajouter un bloc de contenu » et choisir celui que vous souhaitez ajouter.
. Configurer le bloc de contenu quand c'est possible.
. Enfin, déplacer le bloc de contenu dans la colonne des blocs de contenu actifs pour le faire apparaître sur la page d'accueil.

== Configuration

Ci-dessous, vous trouverez tous les contenus de la page d'accueil des guides de configuration et les captures d'écran pour prévisualiser l'affichage sur la page d'accueil.

=== Image principale et CTA

Les blocs de contenu de l'image de bannière et CTA permettent d'afficher dans la page d'accueil de l'espace les informations suivantes :

* *Titre* : le titre de l'espace.
* *Hashtag* : le hashtag de l'espace redirigeant vers la plateforme X.
* *Sous-titre* : le sous-titre de l'espace.
* *Un bouton d'appel à action* : par défaut, le bouton n'est pas affiché. Pour l'activer, cliquer sur l'icône "Modifier" du bloc de contenu.
* *Une image* : l'image téléchargée est affichée avec un filtre d'opacité, pour s'assurer que le texte est lisible.

=== Annonce

Le bloc d'annonce affiche le contenu du champ d'annonce de la configuration de l'espace, dans une boîte bleue d'annonce.
Il permet aux administrateur·ices de mettre en valeur des informations importantes.

image::spaces/landing_page/announcement_front.png[Landing page content Announcement content block]

=== Données principales

Le bloc des données principales affiche la description courte et la description de l'espace et pour les assemblées, le but de l'action et de la composition.

image::spaces/landing_page/main_data.png[Landing page content Main data content block]

=== Étape / Type & durée

Le bloc étape & durée affiche la date de début et de fin de la concertation, et le bloc actuel xref:admin:spaces/processes/phases.adoc[étape] des concertations.
Pour les assemblées, il affiche la durée de l'assemblée.

image::spaces/landing_page/duration.png[Landing page content Phase and duration content block]

=== Métadonnées

Le bloc de métadonnées affiche tous les champs de métadonnées saisis dans la configuration de l'espace.

image::spaces/landing_page/metadata.png[Landing page content Metadata content block]

=== Dernière activité

Le dernier bloc d'activité montre les dernières activités de l'espace, les articles d'actualité, les commentaires, les débats, etc.
Vous pouvez configurer le nombre d'avatars affichés à côté du titre.

=== Documents connexes

Les documents connexes permettent à l'administrateur·ice d'afficher les fichiers xref:admin:spaces/processes/attachments.adoc[PIèces jointes] sur la page d'accuieil, à l'exception des images.

image::spaces/landing_page/documents.png[Landing page content Related documents content block]

=== Images associées

Les images associées permettent à l'administrateur·ice d'afficher les images xref:admin:spaces/processes/attachments.adoc[Pièces jointes] sur la page d'accueil.

image::spaces/landing_page/images.png[Landing page content Related images content block]

=== Rencontres à venir

Les rencontres à venir permettent à l'administrateur·ice d'afficher les prochaines xref:admin:components/meetings.adoc[Rencontres] sur la page d'accueil avec une carte si elles se déroulent à un endroit défini.
Vous pouvez configurer la fonctionnalité pour récupérer les rencontres en cliquant sur le bouton Modifier du bloc de contenu.

image::spaces/landing_page/meetings.png[Landing page content meetings content block]

=== Publications

Le bloc de contenu des messages permet à l'administrateur·ice d'afficher les derniers messages publiés dans l'espace, avec leurs images si elles ont été ajoutées en tant que pièces jointes aux messages.
Vous pouvez configurer la fonctionnalité pour récupérer les messages en cliquant sur le bouton Modifier du bloc de contenu.

image::spaces/landing_page/posts.png[Landing page content posts content block]

=== Propositions

Les propositions permettent d'afficher un échantillon des propositions sur la page d'accueil de l'espace.

image::spaces/landing_page/proposals.png[Landing page content proposals block edition]

Vous pouvez configurer l'ordre d'affichage et la Fonctionnalité propositions en cliquant sur le bouton Modifier du bloc de contenu.

image::spaces/landing_page/proposals_admin.png[Landing page proposals content block]

=== Réalisations

image::spaces/landing_page/results.png[Landing page results content block]

Vous pouvez configurer l'ordre d'affichage et la responsabilisation pour récupérer les réalisations de la fonctionnalité en cliquant sur le bouton Modifier du bloc de contenu.

=== HTML

Le bloc de contenu HTML permet d'ajouter un bloc HTML modifiable sur la page d'accueil et créer ainsi une section personnalisée.
