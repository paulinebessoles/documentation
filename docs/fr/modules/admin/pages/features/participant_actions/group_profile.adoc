= Profils de groupes

== En bref

En tant que participant, il peut être utile de créer un groupe d'utilisateurs pour votre association, votre entreprise, votre club, etc.
Si un administrateur valide votre droit de représenter ce groupe, alors vous pouvez participer dans ce cas au nom
du groupe.

NOTE: L'option de création de groupe n'est disponible que lorsque les groupes sont activés pour l'organisation.
Vous pouvez demander à un administrateur de la plateforme ou l'activer depuis le panneau xref:admin:settings/configuration.adoc[Configuration].

== Configuration

=== Créer un groupe

Pour créer un groupe, cliquez sur "Créer un groupe" dans votre profil public.

image::features/my_profile/profile_create_group.png[Public profile group creation form]

|===
|Champ |Type |Description

|Nom de l'organisation, association, collectif, groupe, etc.

|Pseudo

Le groupe d'utilisateurs est visible sur la plateforme dès sa création. Après sa création, vous serez redirigé vers la page de profil du groupe d'utilisateurs
.

Les pages des profils des groupes d'utilisateurs affichent :

* Membres: Les membres des groupes, et qui en sont les administrateurs.
* Badges : badges du groupe d'utilisateurs.
* Abonnés: Les participants peuvent également suivre les groupes d'utilisateurs.

=== Gérer un groupe

Une fois qu'un groupe a été créé, il est possible d'inviter des participants à devenir membres du groupe.
Un participant a accepté dans ce groupe peut devenir administrateur.

Les actions suivantes peuvent être effectuées par un administrateur de groupe d'utilisateurs :

* Modifier le profil du groupe
* Gérer les membres
* Gérer les administrateurs
* Invite participant
* Quitter le groupe

image::features/my_profile/profile_manage_group.png[Public profile group drop down menu]

==== Gérer les membres

Sur cette page, vous pouvez accepter ou refuser les demandes des participants à rejoindre votre groupe. Vous pouvez aussi promouvoir les membres du groupe au rôle d'administrateur
ou les retirer du groupe.

image::features/my_profile/user_group_manage_participant.png[User groups: manage participants]

==== Gérer les administrateurs

Sur cette page, vous pouvez seulement supprimer les droits d'administrateur du groupe, mais non le status de participant au groupe.

image::features/my_profile/user_group_manage_admins.png[User groups: manage admins]

==== Inviter des participants

Pour inviter une personne à devenir participante, elle doit avoir un compte sur la plateforme.
Les personnes peuvent être invitées à l'aide leur surnom.

image::features/my_profile/user_group_invite_participant.png[User groups: invite a participant]

Elles recevront ensuite une invitation à rejoindre le groupe. Elles peuvent l'accepter ou la refuser.

==== Conversations

Vous pouvez également avoir des conversations en groupe.

image::features/my_profile/user_group_conversation.png[User group: conversations]

=== Participer en tant que groupe

NOTE: Pour que les administrateurs des groupes d'utilisateurs puissent participer en tant que groupe, un administrateur de l'instance doit l'avoir vérifié.

The interest of having an administrator verify the user group is that you can then participate to the platform as a group,
and create debates, proposals, meetings, comments, etc.

Les administrateurs du groupe d'utilisateurs peuvent par exemple commenter au nom de leur groupe.

image::features/my_profile/user_group_make_comment.png[User group: make a comment]

Ils peuvent également choisir d'approuver un débat ou une proposition au nom de leur groupe et/ou en leur propre nom.

image::features/my_profile/user_group_endorse.png[User group: endorse]
