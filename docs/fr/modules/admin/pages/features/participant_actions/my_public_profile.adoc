= Mon profil public

== En bref

Chaque participant a un profil public qui affiche des détails à son sujet et son activité sur la plateforme.
Cette page de profil est accessible à tous les participants et visiteurs.

=== How to find it?

Pour voir le profil public d'un participant, cliquer directement sur son avatar ou son nom d'utilisateur dans l'info-bulle du participant.

image::features/my_profile/profile_pop_up.png[Participant tool-tip]

En tant que participant connecté, vous pouvez accéder à votre profil public à partir de deux endroits :

. En cliquant sur votre avatar dans l'en-tête, puis sur "Mon profil public" dans le menu
. En cliquant sur le lien "Mon profil public" en pied de page, dans la section "Mon compte"

image::features/my_profile/profile_dropdown.png[Frontend user menu dropdown]

=== Onglets de profil

Ici, vous trouverez différents onglets.

NOTE: Les onglets Badges et Groupes ne seront pas visibles si vous ne les avez pas activés dans le panneau xref:admin:settings/configuration.adoc[Configuration].

==== Activité

Cet onglet affiche les activités du participant, à commencer par le contenu le plus récent.

image::features/my_profile/profile_activity.png[Public profile activity tab]

==== Badges

Les badges du profil public montrent quels badges le participant a accumulé et quel est son niveau actuel pour chaque badge.

Des informations complémentaires à propos des badges sont disponibles sur xref:admin:features/social_features/badges.adoc[Badges feature page].

image::features/my_profile/profile_badges.png[Public profile badges tab]

==== Abonnements

La section suivante contient une liste des participants et des groupes que le participant suit actuellement.
En suivant un autre participant ou un groupe sur la plateforme, le participant reçoit des notifications concernant l'activité
de ces participants ou groupes sur la plateforme. Certains participants peuvent également exiger qu'ils soient abonnés
un participant avant de pouvoir échanger des messages privés entre eux.

image::features/my_profile/profile_follows.png[Public profile follows tab]

==== Abonnés

La section « Abonnés » contient une liste des participants et des groupes qui suivent le participant.

image::features/my_profile/profile_followers.png[Public profile followers tab]

==== Groupes

La section Groupes contient tous les groupes auxquels l'utilisateur appartient.

image::features/my_profile/profile_groups.png[Public profile groups tab]

== Configuration

=== Dans le profil d'un autre participant

Un participant connecté peut effectuer trois actions depuis la page du profil public d'un autre participant :

. Lui envoyer un message
. Le suivre
. Le signaler

=== Dans votre propre profil

Les participants connectés peuvent effectuer deux actions depuis leur page de profil public :

. Modifier son propre profil : en cliquant sur ce bouton, ils sont redirigés sur leur xref:admin:features/participant_actions/my_account.adoc[account].
. Créer un groupe : en cliquant sur ce bouton, ils sont redirigés sur le xref:admin:features/participant_actions/group_profile.adoc[group creation form].

NOTE: L'option de création de groupe n'est disponible que lorsque les groupes sont activés pour l'organisation.
