= Notifications

== En bref

Les notifications permettent aux participant·e·x·s de se tenir au courant des dernières nouvelles de la plateforme, des espaces, du contenu, des participant·e·x·s qu'iels suivent, etc.
La plupart des actions sur la plateforme génèrent des notifications.

Les participant·e·x·s peuvent gérer leurs préférences de notification dans leur xref:admin:features/participant_actions/my_account.adoc[compte].

=== How to find it?

En tant que participant·e·x connecté·e·x, vous pouvez accéder à vos notifications depuis deux endroits :

. En cliquant sur votre avatar dans l'en-tête, puis sur "Notifications" dans le menu
. En cliquant sur le lien "Notifications" en pied de page, dans la section "Mon compte"

image::features/notifications/notifications_dropdown.png[Frontend user menu dropdown]

Les notifications apparaissent dans un petit badge à côté de l'avatar de lae participant·e·x, ainsi que dans le menu déroulant.

image:features/notifications/notifications.png[Exemple de page de notifications lorsqu'il y a des notifications]

Lorsqu’il n’y a pas de notifications, ou que lae participant·e·x a marqué toutes les notifications comme lues, un message s’affiche indiquant qu’il n’y a pas de nouvelles notifications.

image:features/notifications/no_notifications_yet.png[Exemple de page de notifications quand il n'y a pas de notifications]
