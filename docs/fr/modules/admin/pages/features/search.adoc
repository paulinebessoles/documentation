= Moteur de recherche

== Understand the basics

*Le moteur de recherche* permet aux participants d'effectuer des recherches sur tout le contenu de la plateforme qui peut être indexé,
de manière générale et spécifique, en recherchant dans un processus participatif spécifique ou à l'intérieur de ses composantes
(propositions, résultats, etc.). , par le biais de recherches avancées.

== How to find it?

Le moteur de recherche est accessible dans l'en-tête de l'instance, à côté du logo de votre plate-forme.

image::features/search/search_input.png[Search input in frontend navigation bar]

Dans la barre de recherche, vous pouvez entrer un mot ou simplement faire une recherche vide pour accéder à la page de résultats de recherche.

image::features/search/search_page.png[Search results page example]

Le moteur de recherche recherche la saisie dans tous les titres et descriptions des ressources.

Vous pouvez également filtrer la recherche en cliquant sur les types de ressources dans la gauche.
