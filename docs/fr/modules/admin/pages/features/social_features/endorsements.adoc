= Soutiens

== En bref

Par le biais d'approbations, les participant·e·x·s peuvent manifester publiquement leur soutien à un contenu particulier, comme une proposition, un débat, etc.

NOTE: Les propositions sont également appelées "J'aime" dans l'interface des participant·e·x·s.

image::features/endorsements/sidebar.png[Endorsement button in sidebar]

== Configuration

=== Activer les approbations

Les administrateur·ices peuvent autoriser les approbations par phase (étape) dans un processus participatif donné, ainsi que définir les autorisations avec les autorisations.

image::features/endorsements/admin.png[Endorsement phase (step) settings]

=== Soutenir / aimer le contenu

Pour approuver le contenu, cliquez sur le bouton "J'aime" sous le contenu.

image::features/endorsements/list.png[List of endorsements]

Lorsqu'unx participant·e·x appartient à un groupe vérifié, iel peut également approuver/aimer ce groupe en sélectionnant l'identité.

image::features/endorsements/modal.png[Select identity modal]

Par défaut, la liste des avances est réduite. Pour voir toutes les personnes en soutiens, vous devez aussi cliquer sur l'icône "+ x".

image:features/endorsements/full_list.png[Liste complète des approbations]

Les participant·e·x·s peuvent également ne pas apprécier le contenu s'iels changent d'avis.
