= Hashtags

== En bref

Les hashtags sont un moyen de catégoriser le contenu globalement sur la plateforme et de le rendre plus consultable à travers ces catégorisations.
Il permet à différents contenus liés d'être découverts sous le même sujet.
Des fonctionnalités similaires sont souvent utilisées sur d'autres plateformes de médias sociaux.

Les hashtags peuvent être définis pour différents enregistrements au sein de la plate-forme, tels que les espaces participatifs.
Ils peuvent également être générés pour le contenu créé par les participants afin de faciliter la découverte du contenu.
Certains composants peuvent également permettre de définir des hashtags suggérés ou des hashtags spécifiques pour tous les enregistrements soumis par les participants.

== Configuration

Les participants peuvent ajouter des hashtags dans leurs commentaires, débats, propositions, etc.
En cliquant sur un hashtag dans un commentaire ou un autre contenu, le participant est redirigé vers la recherche globale.
