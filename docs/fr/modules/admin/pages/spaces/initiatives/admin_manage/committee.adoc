= Membres du comité

== Understand the basics

NOTE: L'explication du comité de promotion est disponible sur xref:admin:spaces/initiatives/author_manage/committee.adoc[Comité du promoteur].

Les membres du comité pourraient être nécessaires, selon les règles établies pour les types d'initiative.
Les participants peuvent les gérer à partir de leur formulaire d'édition d'initiative, mais les administrateurs peuvent également accéder à la liste des membres du comité
.

== Configuration

=== Gérer les membres du comité

Pour accéder à la liste des membres du comité, cliquez dans le panneau de navigation secondaire du point « Membres du comité ».

image::spaces/initiatives/initiative_members_admin.png[Initiative committee members list in administration]

Là, l'auteur peut voir et copier le lien d'invitation d'initiative, et voir la liste des membres du comité.
