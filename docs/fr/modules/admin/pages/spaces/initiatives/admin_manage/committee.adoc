= Membres du comité

== En bref

NOTE: L'explication du comité promotion est disponible sur xref:admin:spaces/initiatives/author_manage/committee.adoc[Comité promoteur].

Les membres du comité pourraient être nécessaires, selon les règles établies pour les types d'initiative.
Les participant·e·x·s peuvent les gérer à partir de leur formulaire de modification d'initiative, mais les administrateur·ices peuvent également accéder à la liste des membres du comité
.

== Configuration

=== Gérer les membres du comité

Pour accéder à la liste des membres du comité, cliquer dans le panneau de navigation secondaire sur « Membres du comité ».

image::spaces/initiatives/initiative_members_admin.png[Initiative committee members list in administration]

Là, l'auteur·ice peut voir et copier le lien d'invitation à l'initiative, et voir la liste des membres du comité.
