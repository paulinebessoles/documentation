= Concertations

== En bref

Une concertation est une séquence d'activités participatives dans le but de définir et de prendre une décision sur un sujet donnée.
Par exemple, vous pourriez avoir une concertation comme suit :

. Répondre d'abord à une enquête pour recueillir l'opinion des participant·e·x·s ;
. Puis lancer des propositions ;
. Discuter en face à face ou en réunion virtuelle ;
. Enfin en les priorisant avant leur mise en œuvre.

NOTE: Vous voulez créer un processus participatif ? Vérifiez l'article xref:admin:spaces/processes/process_creation.adoc[Créer une concertation] .

Les exemples de concertations sont : une concertation pour l'élection des membres du comité (où les candidatures sont présentées pour la première fois,
alors débattu et finalement une candidature est choisie), des budgets participatifs (où des propositions sont faites, évalué économiquement,
et voté avec l'argent disponible), une concertation de planification stratégique, l'élaboration collaborative d'un règlement ou d'une norme,
la conception d'un espace urbain ou la production d'un plan de politique publique.

=== How to find it?

Pour accéder au panneau des concertations, aller dans le panneau d'administration et cliquer sur "Concertations" dans la barre de navigation d'administration
.

image::spaces/processes/processes_menu.png[Pages: Access processes panel]

Ici, vous pouvez gérer des concertations et des groupes de concertation.

== Configuration

=== Gérer les concertations

image::spaces/processes/processes_list.png[Participatory processes list]

Sur cette page, vous pouvez gérer les concertation et xref:admin:spaces/processes/groups.adoc[groupes de concertation].

Vous pouvez trouver toutes les concertations listées dans le panneau d'administration, avec :

* *Titre* : titre de la concertation.
* *Créé le* : date de création de la concertation.
* *Privée* : si la concertation est ouverte à touxtes les participant·e·x·s ou seulement à certain·e·x·s participant·e·x·s.
* *Publiée* : si la concertation est publiée ou non.

Il est possible de filtrer et trier cette liste en cliquant sur "Filtrer" et sur les en-têtes des colonnes.
Vous pouvez également rechercher des concertations par titre dans la barre de recherche.

image::spaces/processes/processes_list_filter.png[Participatory processes list filtered]

=== Importer une concertation

Pour importer une concertation, cliquer sur "Gérer" en haut à droite de la page.

image::spaces/processes/import_menu.png[Participatory processes import menu]

Cela vous redirige vers la page d'importation, où vous devez remplir les champs suivants :

* Titre : Le titre que vous souhaitez donner à la concertation.
* Identificateur d'URL : Le « slug » pour la concertation.
* Ajouter un document : Ici vous devez charger le fichier JSON de la concertation obtenu par mail lors de l'exportation.

Vous pouvez ensuite sélectionner le type de données que vous souhaitez importer à partir du fichier.

* Étapes d'importation : Importation de toutes les étapes présentes dans le fichier original.
* Catégories d'importation : Importation de toutes les catégories présentes dans le fichier original.
* Importer des pièces jointes : Importation des pièces jointes de la concertation originale.
* Importer des fonctionnalités : Importation de toutes les fonctionnalités et leur configuration à partir du fichier original, sans importer leur contenu.
