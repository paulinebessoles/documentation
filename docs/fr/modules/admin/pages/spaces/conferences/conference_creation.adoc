= Création de la conférence

== En bref

Les conférences permettent aux administrateurs de configurer des composants dans le contexte d'un événement, en général en articulation avec un programme.
Pour commencer à créer une conférence, lorsque dans la liste des conférences, cliquez sur le bouton "Nouvelle conférence" en haut à droite de la page.

Le formulaire de création de la conférence est divisé en différentes sections détaillées dans les sections ci-dessous.

La page d'accueil des conférences est construite par défaut et n'est pas encore configurable.

=== Création du programme

For creating the program of the conference you need to create a new xref:components/meetings.adoc[Meeting component] with the
dates and times of the different talks, workshops, sessions, etc. After publishing the meetings, the program button
is visible in the landing page.
