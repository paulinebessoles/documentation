= Haut-parleurs

== En bref

Les orateurs sont les personnes qui parlent à la conférence. They can be linked to meetings, so participants can
see who is going to speak in each item of the program.

image::spaces/conferences/speakers_frontend.png[Conferences speakers frontend]

== Configuration

=== Gérer les orateurs

Pour accéder à la configuration des haut-parleurs, cliquez dans le panneau de navigation secondaire de l'élément "Haut-parleurs".

image:espaces/conférences/haut-parleurs.png[conférenciers de conférences]

Vous pouvez rechercher des haut-parleurs grâce à la barre de recherche.

=== Créer un nouveau haut-parleur

Pour créer un nouveau haut-parleur, cliquez sur le bouton "Nouveau haut-parleur" en haut à droite de la page.

image::spaces/conferences/new_speaker.png[Conferences speakers]
