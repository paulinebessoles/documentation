= Conférencier·e·x·s

== En bref

Les conférencier·e·x·s sont les personnes qui donnent la conférence. Iels peuvent être lié·e·x·s à des rencontres pour que les participant·e·x·s puisent être au courant des personne qui interviennent durant la conférence dans le preogramme.

image::spaces/conferences/speakers_frontend.png[Conferences speakers frontend]

== Configuration

=== Gérer les conférencier·e·x·s

Pour accéder à la configuration des conférencier·e·x·s, cliquer dans le panneau de navigation secondaire "Conférencier·e·x·s".

image:espaces/conférences/haut-parleurs.png[conférencier·e·x·s]

Vous pouvez rechercher des conférencier·e·x·s grâce à la barre de recherche.
