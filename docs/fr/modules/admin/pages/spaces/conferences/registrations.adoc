= Inscriptions

== En bref

En tant qu'administrateur·ice d'une conférence, vous pouvez gérer les inscriptions pour la conférence directement à partir de la plateforme.

Pour ce faire, vous pouvez utiliser ces sections d'administration :

* xref:admin:admin:spaces/conferences/registrations/types.adoc[**Types d'inscriptions**]: pour définir les moyens de s'inscrire.
* xref:admin:admin:espaces/conférences/registrations/users.adoc[**Inscriptions d'utilisateurs**]:pour confirmer les inscriptions.
* xref:admin:admin:spaces/conferences/registrations/invites.adoc[**Invité·e·x·s**]: inviter des participant·e·x·s à la conférence.
* xref:admin:admin:espaces/conférences/inscriptions/certificates.adoc[**Attestations de présence**]:configurer et envoyer les attestation
de présence au format PDF aux participant·e·x·s de la conférence.
