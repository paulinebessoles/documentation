= Partenaires

== En bref

Les partenaires sont les commanditaires ou les soutiens d'une conférence.
Si vous ajoutez des partenaires dans l'administration, alors ils sont listés sur la page d'accueil de la conférence.

image:espaces/conferences/partners_frontend.png[Gestion des partenaires de la conférence]

== Configuration

=== Gérer les partenaires

Pour accéder à la configuration des partenaires, cliquer dans le panneau de navigation secondaire sur "Partenaires".

image::spaces/conferences/partners.png[Conferences partners]

=== Créer un nouveau partenaire

Pour créer un nouveau partenaire, cliquer sur le bouton "Nouveau partenaire" en haut à droite de la page.

image::spaces/conferences/new_partner.png[Conferences partners form]

.Formulaire de nouveau partenaire
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Nom
|Obligatoire
|Le nom du partenaire.

|Type de partenaire
|Obligatoire
|Choisir l'une de ces options : lae promoteur·ice principal ou lae collaborateur·ice.

|Rang d'affichage
|Facultatif
|Position par rapport aux autres partenaires. Avoir un nombre inférieur signifie un rang d'affichage plus élevé. Par exemple, si vous avez deux partenaires, "Partenaire a" de poids "10" et "Partenaire b" de poids "0", alors "Partenaire b" apparait en premier dans la liste.

|Lier
|Facultatif
|Lien vers le site du partenaire, la biographie, les médias sociaux, etc.

|Logo
|Obligatoire
|Orientation pour l'image: De préférence une image de paysage qui n'a pas de texte. Le service récolte l'image. Taille maximale du fichier : 5 Mo
|===
