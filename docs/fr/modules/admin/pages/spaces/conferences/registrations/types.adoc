= Types d'enregistrement

== En bref

Les inscriptions sont l'une des particularités des espaces de conférence, qui permettent à des participant·e·x·s de s'inscrire à une conférence entière, à des jours choisis seulement ou à des moments du programme de la conférence, plutôt que de s'inscrire à des rencontres une par une.

NOTE: Si vous voulez activer l'inscription, vous devez définir et publier au moins un type d'inscription.

Les administrateur·ices peuvent configurer ces différents types de "tickets" dans la section types d'inscription :

. Définir les types d'inscription
. Activer les inscriptions quand elles sont prêtes
. Les participant·e·x·s sont alors en mesure de s'inscrire et de choisir le type d'inscription

Une fois les inscriptions activées, les participant·e·x·s peuvent cliquer sur un bouton pour s'inscrire à la conférence dans la page d'accueil
et dans l'en-tête de la conférence.

== Configuration

=== Gérer les types d'inscription

Pour accéder à la configuration des types d'inscription, cliquer dans le panneau de navigation secondaire sur "Liens médias".

image::spaces/conferences/registration_types_backend.png[Manage registration types]

=== Créer un nouveau type d'inscription

Pour créer un nouveau type d'inscription, cliquer sur le bouton "Nouveau type d'inscription" en haut à droite de la page.

image::spaces/conferences/new_registration_type.png[Create a new registration type form]

.Créer un nouveau type d'inscription
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Titre
|Obligatoire
|Le titre de ce type d'inscription.

|Rang d'affichage
|Obligatoire
|La position à laquelle se trouve le type en relation avec les autres types. Avoir un nombre inférieur signifie un rang d'affichage plus élevé. Par exemple, si vous avez deux types, "Jour 1" de poids "10" et "Jour 2" de poids "0" alors "Jour 2" apparait en premier dans le listage.

|Description
|Obligatoire
|Description pour ce type d'inscription.

|Prix
|Facultatif
|Définissez un prix pour ce type d'inscription. Si aucun prix n'est défini, alors c'est gratuit.

|Sélectionner les rencontres de la conférence
|Facultatif
|Les rencontres auxquelles ce type d'inscription donne accès.
|===

=== S'inscrire en tant que participant·e·x

Si lae participant·e·x n'est pas connecté·e·x, eiel a la possibilité de se connecter ou créer un compte en cliquant sur "S'inscrire",
et peuvent ainsi consulter les types d'inscription.

image:espaces/conférences/registration_types_frontend_wo_user.png[Sélectionnez le type d'inscription sans compte utilisateur·e·x]

Si lae participant·e·x est connecté·e·x, iel peut s'inscrire directement en cliquant sur "S'inscrire".

image::spaces/conferences/registration_types_frontend_w_user.png[Select registration type with user account]

En cliquant sur le bouton S'inscrire pour un type d'inscription, un pop-up apparait avec les délais de l'inscription du formulaire de configuration de la conférence.

image::spaces/conferences/registration_pop_up.png[Pop up with registration terms]
