= Invitations

== En bref

Les administrateur·ices peuvent inviter les participant·e·x·s à participer à la conférence.
Iels peuvent inviter des participant·e·x·s existant·e·x·s, détenant un compte sur la plateforme, ainsi que des personnes sans compte.
Lorsqu'unx participant·e·x est invité·e·x, un mail est envoyé à l'adresse de son compte, l'encourageant à s'inscrire à la conférence.

== Configuration

=== Gérer les invitations

Pour accéder à la configuration des invitations, cliquer dans le panneau de navigation secondaire sur "Invitations".

image::spaces/conferences/invites.png[Conferences user invites in backend]

Vous pouvez filtrer les invitations et rechercher par nom ou par adresse email.

=== Envoyer une invitation

Pour inviter un participant, cliquez sur le bouton "Inviter un participant" en haut à droite de la page.

image::spaces/conferences/invite_participant_form.png[Invite a participant form]
