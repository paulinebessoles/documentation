= Invitations

== En bref

Les administrateurs peuvent inviter les participants à participer à la conférence.
Ils peuvent inviter des participants existants, avec un compte sur la plateforme, ainsi que des personnes sans compte.
Inviter un participant envoie un e-mail à l'adresse e-mail du compte, encourageant le participant à s'inscrire à la conférence.

== Configuration

=== Gérer les invitations

Pour accéder à la configuration des invitations, cliquez dans le panneau de navigation secondaire de l'élément "Invitations".

image::spaces/conferences/invites.png[Conferences user invites in backend]

Vous pouvez filtrer les invitations et rechercher par nom d'invité ou par email.

=== Envoyer une invitation

Pour inviter un participant, cliquez sur le bouton "Inviter un participant" en haut à droite de la page.

image::spaces/conferences/invite_participant_form.png[Invite a participant form]
