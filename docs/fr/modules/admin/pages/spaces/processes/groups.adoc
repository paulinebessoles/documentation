= Groupes de processus

== Understand the basics

Un groupe de processus peut être utile pour subdiviser les grands processus participatifs en sous-processus selon des critères.
It can also allow administrators to organize an instance with a lot of processes, to gather together the ones related to a
specific type of process, or a specific scope, like participatory budgeting annual editions, mobility planning processes, etc.

== Configuration

=== Gérer les groupes de processus

Pour accéder à la page de gestion des groupes de processus, cliquez sur l'onglet "Groupes de processus" dans la page d'index des processus.
Vous y trouverez la liste des groupes de processus.

Pour créer un nouveau groupe de processus, cliquez sur le bouton "Nouveau groupe de processus" en haut à droite de la page.

=== Blocs de contenu de la page d'accueil

Les groupes de processus sont principalement des pages de destination, regroupant des processus participatifs plus complexes.
Cette page d'accueil fonctionne avec différents blocs de contenu, tout comme la page d'accueil générale de la plateforme.
Cela signifie que vous pouvez activer différentes sections de la page d'accueil et configurer certaines de ces sections.

Pour configurer une page d'accueil de groupe de processus, lors de l'édition du groupe de processus, cliquez sur le bouton "Gérer" en haut à droite de la page,
et cliquez sur le lien de menu "Page d'accueil".

image::spaces/processes/processes_group_landing.png[Participatory process group landing page]

Ci-dessous, vous trouverez tous les guides de configuration des blocs de contenu et les captures d'écran de la façon dont ils sont affichés sur la page de destination.

NOTE: S'il n'y a pas de contenu à afficher dans un bloc de contenu ou s'il n'est pas configuré, il ne sera pas affiché sur la page d'atterrissage.

==== Titre et description

Le bloc de contenu Titre et description permet d'afficher :

* *Titre* : le titre du groupe de processus.
* *Description* : la description du groupe de processus.
* *Nombre de processus associés* : le nombre de processus participatifs publiés liés à ce groupe de processus.
* *Portée* : la portée du groupe de processus.
* *Hashtag* : le hashtag, redirection vers les médias sociaux X.
* *Site Web* : un lien vers le site Web.

image::spaces/processes/processes_group_landing_title.png[Participatory process group landing page title and description block]

image::spaces/processes/processes_group_landing_metadata.png[Participatory process group landing page metadata block]

==== Image principale et bouton d'action

Le bloc image héros et contenu CTA permet d'afficher :

* *Titre* : le titre du groupe de processus.
* *un hashtag* : le hashtag, redirigeant vers les médias sociaux X.
* *un bouton d'appel à l'action*: activez-le en cliquant sur l'icône "Modifier" sur le bloc de contenu de la page d'accueil.
* *une image*: téléchargez une image pour ce bloc en cliquant sur l'icône "Modifier" sur le bloc de contenu de la page d'accueil.

image::spaces/processes/processes_group_landing_hero.png[Participatory process group landing page hero block]

==== Propositions

Le bloc de contenu des propositions permet d'afficher une sélection de propositions provenant des différents processus participatifs du groupe de processus.
Vous pouvez configurer comment les propositions sont triées par ordre le plus récent ou le plus aléatoire en cliquant sur l'icône "Modifier" sur le bloc de contenu de la page d'accueil.

image::spaces/processes/processes_group_landing_proposals.png[Participatory process group landing page proposals block]

==== Résultats

Le bloc de contenu Résultats permet d'afficher une sélection des résultats des différents processus participatifs.
Vous pouvez configurer la façon dont les résultats sont triés par le plus récent ou au hasard en cliquant sur l'icône "Modifier" sur le bloc de contenu de la page d'accueil.

image::spaces/processes/processes_group_landing_results.png[Participatory process group landing page results block]

==== Réunions à venir

Le bloc de contenu des réunions à venir permet d'afficher une sélection de réunions des différents processus participatifs.
Vous pouvez configurer la façon dont les résultats sont triés par le plus récent ou au hasard en cliquant sur l'icône "Modifier" sur le bloc de contenu de la page d'accueil.
S'il n'y a pas de réunions à venir, ce bloc montre les réunions passées.

image::spaces/processes/processes_group_landing_events.png[Participatory process group landing page events block]

==== Processus participatifs

Le bloc de contenu du processus participatif affiche une grille de processus participatifs sur la page d'atterrissage.
Vous pouvez configurer si seulement les processus actifs seront affichés en cliquant sur l'icône "Modifier" sur le bloc de contenu de la page de destination.

image::spaces/processes/processes_group_landing_processes.png[Participatory process group landing page processes block]

==== Blocs HTML

Le bloc contenu du bloc HTML permet d'ajouter un bloc HTML modifiable sur la page d'atterrissage, et donc de créer une section personnalisée.
