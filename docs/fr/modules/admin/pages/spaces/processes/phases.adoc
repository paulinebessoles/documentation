= Étapes

== En bref

Une concertation est une séquence d'étapes ordonnées dans le temps. Selon l'étape de la chronologie du processus sur laquelle vous
êtes, vous pouvez alors activer et configurer les fonctionnalités dont vous avez besoin pour chaque étape.

Idéalement, les Étapes d'une Concertation sont toujours les mêmes et ne sont pas modifiées, pour que les participant·e·x·s puissent connaitre
les règles et les délais, et peuvent ainsi s'organiser pour savoir quand il serait préférable de participer à ce processus.

Par exemple, la structure standard d'une concertation pourrait être la suivante :

. Présentation
. Diagnostic
. Création et débat de propositions
. Priorisation des propositions
. Évaluation des propositions
. Suivi

Les participant·e·x·s ont accès au lien des différentes étapes de la concertation dans le bloc de contenu de la page d'accueil "Étape et durée", si activé par unx administrateur·ice dans la xref:admin:spaces/processes/landing_page.adoc[Page d'accueil] de la concertation.

image::spaces/processes/process_phases_block.png[Block with processes phases link]

En cliquant sur le lien pour l'étape active, il est possible consulter la description complète et l'explication des étapes dans un pop-up.

image::spaces/processes/process_phases.png[Process phases explanation]

== Configuration

=== Gérer les étpaes

Pour accéder à la configuration des étapes, cliquer dans le panneau de navigation secondaire sur "Étapes".

image::spaces/processes/process_phases_list.png[Phases of a participatory process]

=== Actions

.Actions
[cols="10h,20h,~,~"]
|===
|Icône |Nom |Définition |Condition

|image:icons/action_edit.png[Modifier l'icône]
|Editer
|Modifier cette étape.
|Toujours disponible

|image:icons/action_activate.png[Icône Activer]
|Activer
|Pour faire de cette étape l'étape active.

|image:icons/action_delete.png[Supprimer l'icône]
|Supprimez

=== Créer une nouvelle étape

Pour créer une nouvelle étape, cliquer sur "Nouvelle étape" en haut à droite de la page.

image::spaces/processes/process_phases_new_form.png[Create a new participatory process phase form]

.Formulaire de création d'étape de concertation
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Titre
|Obligatoire
|Le titre de cette étape, affiché dans le bloc de contenu des étapes et sur les cartes de concertation lorsque l'étape est active.

|Description
|Facultatif
|Une longue description expliquant en quoi consiste cette étape.

|Date de début
|Facultatif
|La date de début de cette étape de concertation. Format attendu : jj/mm/aaaa

|Date de fin
|Facultatif
|La date de fin de cette étape de concertation. Format attendu : jj/mm/aaaa

|Chemin de l'appel à action
|Facultatif
|Déprécié

|Texte de l'appel à action
|Facultatif
|Déprécié
|===

CAUTION: La configuration de l'ATC à travers l'étape de la concertation est obsolète dans le nouveau design.
Maintenant cela se fait dans la page d'atterrissage, en utilisant le bloc de contenu Héros et image.
