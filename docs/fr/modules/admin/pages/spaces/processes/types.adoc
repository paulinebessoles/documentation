= Types de concertation

== En bref

Dans certains cas, il est nécessaire d'organiser des concertations par plus que les seuls xref:spaces/processes/groups.adoc[Groupes de concertation].
C'est là que les types de concertation peuvent être utiles, et ajouter un niveau de catégorisation.
Les participant·e·x·s sont alors en mesure de filtrer les concertations par les types de concertation que vous avez créés.

NOTE: Les types de concertation fonctionnent de la même façon que les xref:admin:spaces/assemblies.adoc#_assemblies_types[Types d'assemblée].

== Configuration

=== Gérer les types de concertation

Pour accéder à la liste des types de concertation, cliquer sur « Gérer » en haut à droite de la page, puis cliquer sur « Types de concertation » dans le menu.

image::spaces/processes/process_types_list.png[Backend process types menu item]

=== Créer un type de concertation

Cliquer sur "Nouveau type de concertation" en haut à droite de la page.

image:spaces/processes/process_types_form.png[Formulaire de création de type de concertation]

.Formulaire de création de type de concertation : Informations générales

|===
|Champ |Type |Description

|Titre
|Obligatoire

|===

=== Ajouter un type de concertation à une concertation

Pour relier les concertations aux types de concertation, dans la page de modification d'une concertation, sélectionner le type de concertation sous la section intitulée Autre.

image::spaces/processes/process_types_edit_process.png[Choose a process type when editing a process]

=== Filtrer par types de concertation

S'il y a au moins un type de concertation défini et lié à une concertation, un filtre est affiché à gauche de la liste des concertations
dans l'interface participant·e·x, permettant le filtrage des concertations.

image::spaces/processes/process_types_filter.png[Process types filter in the processes index]
