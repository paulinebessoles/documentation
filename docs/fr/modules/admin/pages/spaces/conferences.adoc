= Conférences

== En bref

Une conférence est une série de rencontres organisées dans un programme, listant un certain nombre de personnes invitées intervenantes, ainsi qu'exposant des champs d'information
typiques des grands congrès ou événements sociaux (inscription, liste des organisations qui soutiennent ou parrainent l'événement
, etc.) Une conférence peut être un événement pertinent pour une organisation et ses membres, ou se dérouler dans le cadre d'un processus participatif
ou suivre une consultation.

Lors des conférences, les fonctionnalités des rencontres sont affichées sous forme de programme, les jours et les heures étant énumérés comme dans un programme d'événement.

=== How to find it?

Pour accéder au panneau Conférences, aller dans le panneau d'administration et cliquer sur "Conférences" dans la barre de navigation d'administration.
.

image::spaces/conferences/menu.png[Conferences menu]

Là, vous pouvez gérer toutes les conférences existantes.

== Configuration

=== Gérer les conférences

image::spaces/conferences/manage_conferences.png[Conferences list in administration panel]

Vous pouvez trouver toutes les conférences listées dans le panneau d'administration, avec :

* *Titre* : titre de la conférence.
* *Créée le* : date de création de la conférence.
* *Publiée* : si la conférence est publiée ou non.

Il est possible de filtrer et trier cette liste en cliquant sur "Filtrer" et sur les en-têtes des colonnes.
Vous pouvez également rechercher des conférences par titre dans la barre de recherche.
