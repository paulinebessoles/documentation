= Créer une assemblée

== En bref

Les assemblées permettent aux administrateur·ices de configurer des fonctionnalités dans un espace de longue durée, avec des membres définis. Pour créer une assemblée, sur le lien :https://docs.decidim.org/fr/develop/admin/spaces/assemblies#_manage_assemblies[liste des assemblées], cliquer sur "Nouvelle assemblée" en haut à droite de la page.

Le formulaire de création d'assemblée est divisé en différentes sections qui sont détaillées ci-dessous.

Une fois une assemblée créée, il est possible de voir toutes les options de l'assemblée disponibles dans le panneau de navigation secondaire :

* xref:admin:admin:spaces/assemblies/assembly_creation.adoc[À propos de cette assemblée] (Ici vous pouvez accéder et modifier tous les champs du formulaire de création.)
* xref:admin:espace/assemblies/landing_page.adoc[Page d'accueil]
* xref:admin:spaces/assemblies/components.adoc[Components]
* xref:admin:spaces/assemblies/categories.adoc[Categories]
* xref:admin:spaces/assemblies/attachments.adoc[Attachments]
* xref:admin:spaces/assemblies/members.adoc[Members]
* xref:admin:spaces/assemblies/admins.adoc[Assembly admins]
* xref:admin:spaces/assemblies/private_participants.adoc[Private participants]
* xref:admin:spaces/assemblies/moderations.adoc[Moderations]

=== Durée

Cette section permet de fournir des informations sur les dates importantes de l'assemblée.

image::spaces/assemblies/assemblies_new_form_duration.png[New assembly form duration section]

.Formulaire pour la création d'une nouvelle assemblée : Durée
[cols="20h,10h,~,~"]
|===
|Champ |Type |Description |Visibilité

|Date de création
|Facultatif
|Quand cette assemblée a été créée. Vous avez un widget calendrier pour choisir la journée. Format attendu : jj/mm/aaaa.
|Visible dans le bloc de la page d'accueil "Dates méta-données".

|Inclus le
|Facultatif
|La date à laquelle cette assemblée a été ajoutée à l'instance. Elle ne doit pas nécessairement être la même que la date de création. Format attendu : jj/mm/aaaa.
|Visible dans le bloc de la page d'accueil "Dates méta-données".

|Durée
|Facultatif
|Si la durée de cette assemblée est limitée, sélectionnez la date de fin. Dans le cas contraire, elle apparaît comme indéfinie. Format attendu : jj/mm/aaaa.
|Visible dans le bloc de la page d'accueil "Type et durée".

|Date de clôture
|Facultatif
|Lorsque cette assemblée est clôturée. Format attendu : jj/mm/aaaa.
|Visible dans le bloc de la page d'accueil "Dates méta-données".

|Raison de la date de clôture
|Facultatif
|Pourquoi cette assemblée est clôturée.
|Non affiché sur la page d'accueil.
|===

=== Visibilité

Cette section permet aux administrateur·ices de configurer le placement du processus dans les différentes pages et leur publicité.

image::spaces/assemblies/assemblies_new_form_visibility.png[New assembly form visibility section]

.Formulaire pour la création d'une nouvelle assemblée : Visibilité
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Assemblée parente
|Facultatif
|Les assemblées à laquelle cette assemblée appartient. Les assemblées peuvent être imbriquées les unes aux autres.

|Surligné
|Facultatif
|Cocher cette case pour que l'assemblée ait plus de visibilité dans la liste publique des Assemblées. Ce sera également visible dans la configuration du contenu du bloc "Assemblées mise en avant" xref:admin:settings/homepage.adoc[Page d'accueil].

|Espace privé
|Facultatif
|Vérifier si cette assemblée ne doit être accessible que par xref:admin:spaces/assemblies/private_participants.adoc[Participant·e·x·s privé·e·x·s].

|Est transparent
|Facultatif
|Vérifier si cette assemblée est privée mais doit être visible par touxtes les participant·e·x·s. Cela signifie que les autres participant·e·x·s
ou les visiteur·euses peuvent le voir, mais ne peuvent pas y participer.
|===
