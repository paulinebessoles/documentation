= Membres de l'assemblée

== En bref

Les membres de l'Assemblée permettent de montrer touxtes les participant·e·x·s qui appartiennent à une assemblée donnée, que ce soit des personnes ou des groupes.
Il s'agit d'une caractéristique spécifique à l'espace des assemblées, fait pour améliorer la transparence des assemblées de l'organisation.

image::spaces/assemblies/assembly_members.png[Assembly members]

== Configuration

=== Gérer les membres de l'assemblée

Pour accéder à la liste des membres de l'assemblée, cliquer dans le panneau de navigation secondaire sur "Membres".

image::spaces/assemblies/assembly_members_list.png[Assembly members administration list]

=== Créer un membre de l'assemblée

Pour créer un nouveau membre de l'assemblée, cliquez sur le bouton "Nouveau membre de l'assemblée" en haut à droite de la page.

image::spaces/assemblies/assembly_members_new_form.png[Assembly members new form]

.Formulaire de nouveau membre de l'assemblée
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Type de participant·e·x
|Obligatoire
|Si le membre de cette assemblée a déjà un compte sur l'instance ou non.

|Nom complet
|Requis.
|Champ qui s'affiche si "non participant·e·x" a été sélectionné.

|Avatars
|Facultatif.
|Champ qui s'affiche si "non participant·e·x" a été sélectionné.

|Utilisateur·ice ou groupe
|Requis si c'est unx participant·e·x existant·e·x.
|Taper au moins trois caractères pour rechercher le compte de lae participant·e·x.

|Position
|Obligatoire
|Choisir parmi président·e·x, vice-président·e·x, secrétaire ou autre. Le cas échéant, vous pouvez compléter le titre du poste.

|Rang d'affichage
|Facultatif
|La position des membres en relation avec les autres membres. Avoir un nombre inférieur signifie un rang d'affichage plus élevé.
Par exemple, si vous avez deux membres, "Membre a" de poids "10" et "Membre b" de poids "0" alors "Membre b" s'affiche en premier dans le listage.

|Date de désignation
|Obligatoire
|Lorsqu'une personne ou un groupe devient membre de cette assemblée. Format attendu : jj/mm/aaaa.

|Date de fin
|Facultatif
|Date à laquelle cette personne ou ce groupe cessé d'être un membre de l'assemblée. Format attendu : jj/mm/aaaa.

|Genre
|Facultatif
|À quel genre cette personne s'identifie le plus.

|Date d'anniversaire
|Facultatif
|Quand cette personne est-elle née. Format attendu : jj/mm/aaaa.

|Lieu de naissance
|Facultatif
|Où cette personne est-elle née.
|===
