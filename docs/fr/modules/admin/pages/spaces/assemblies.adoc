= Ruches

== En bref

An assembly is a group of members of an organization who meet periodically to make decisions about a specific area or
scope of the organization.

Les assemblées peuvent tenir des réunions, consulter leurs membres, être privées ou ouvertes à tous.

Examples of assemblies are: a general assembly meeting once a year to define the organisation's main lines of action as well as its
executive bodies by vote, an equality advisory council meeting regularly to make proposals on how to improve
gender relations in the organisation, an evaluation commission meeting every month to monitor a process, or a guarantee
body collecting incidents, abuses, or proposals to improve decision-making procedures.

NOTE: The main difference between xref:admin:spaces/processes.adoc[participatory processes] and assemblies is that assemblies don't
have phases, meaning that they don't have timelines.

=== How to find it?

Pour accéder au panneau Ruches, allez dans le panneau d'administration et cliquez sur l'élément "Ruches" dans la barre de navigation d'administration
.

image::spaces/assemblies/assemblies_menu.png[Pages: Access assemblies panel]

Là, vous pouvez gérer les assemblages et les types d'assemblages.

== Configuration

=== Gérer les assemblées

image::spaces/assemblies/assemblies_list.png[Assemblies list]

Sur cette page, vous pouvez gérer les assemblages et xref:admin:spaces/processes/groups.adoc[types d'assemblées].
Vous pouvez trouver tous les assemblages dans la liste dans le panneau d'administration, avec:

* *Titre* : titre de l'assemblée.
* *Créé à* : date de création de l'assemblée.
* *Privé* : si l'assemblée est ouverte à tous les participants ou à certains participants.
* *Publié* : si l'assemblage est publié ou non.

Il est possible de filtrer et trier cette liste en cliquant sur "Filtrer" et sur les en-têtes des colonnes.
Vous pouvez également rechercher des assemblages par titre dans la barre de recherche.

image::spaces/assemblies/assemblies_list_filter.png[Assemblies list filtered]

=== Importer un assemblage

Pour importer une assemblée, cliquez sur le bouton "Gérer" en haut à droite de la page.

image::spaces/assemblies/import_menu.png[Assemblies import menu]

Vous êtes redirigé vers la page d'importation, où vous devez remplir ces champs :

* Titre : Le titre que vous souhaitez donner à l'assemblée.
* Identifiant d'URL : Le « slug » pour l'assemblage.
* Ajouter un document : Ici vous devez charger le fichier JSON de l'assembly obtenu par email lors de l'exportation.

Vous pouvez ensuite sélectionner le type de données que vous souhaitez importer à partir du fichier.

* Catégories d'importation : Importe toutes les catégories présentes dans le fichier original.
* Importer des pièces jointes : Importer des pièces jointes de l'assemblage original.
* Importer des composants : Importe tous les composants et leur configuration à partir du fichier original, sans importer leur contenu.
