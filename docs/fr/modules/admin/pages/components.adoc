= Fonctionnalités

== En bref

Grâce aux fonctionnalités, unx administrateur·ice peut activer et configurer différents mécanismes de participation.
Toutes les fonctionnalités sont disponibles dans tous les différents xref:admin:spaces.adoc[espaces].

Lors de la création d'un espace participatif, l'espace, par défaut, n'aura aucune fonctionnalité.

include::partial$components_list.adoc[]

=== How to find it?

Les fonctionnalités sont atteignables dans chaque xref:admin:spaces.adoc[espace], en passant par la barre de navigation secondaire de l'administration.

image::components/components_index_back.png[Component index backend in an assembly]

POur créer une fonctionnalité, cliquer sur "Ajouter fonctionnalité" et sélectionner le type de fonctionnalité à créer.

Une fois la publication de fonctionnalités entamée, les participant·e·x·s voient les liens des différentes fonctionnalités apparaitre dans l'espace du bloc de contenu "Données principales" dans la page d'accueil, dans le moteur de recherche général et dansde bloc de contenu de la page d'accueil dédiée.

image::components/components_index_front.png[Component index frontend in an assembly]

== Configuration

=== Gérer les fonctionnalités

Sur cette page, il est possible de gérer les fonctionnalités et d'en créer de nouvelles.

Les fonctionnalités de l'espace se trouvent dans la liste, avec :

* *Nom de la fonctionnalité* : le titre de la fonctionnalité choisi à sa création et modifiable.
* *Le type de fonctionnalité* : le type de la fonctionnalité.
* *Le secteur de la fonctionnalité* : le xref:admin:scopes.adoc[secteur] de la fonctionnalité.
