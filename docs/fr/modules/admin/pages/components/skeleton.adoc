== Configuration

Pour créer une fonctionnalité de blog, aller d'abord dans [espace participatif] à l'endroit de création souhaité.

image::components/skeleton/component_skeleton.png[Add component button]

Ensuite :

. Cliquer sur l'élément "Fonctionnalités" dans le menu secondaire
. Cliquer sur "Ajouter une fonctionnalité"
. Sélectionnez "Skeleton"
. Remplir le formulaire de création de fonctionnalité

image:components/skeleton/component.png[Ajouter un formulaire de composant]


.Ajouter un composant : formulaire de squelette
|===
|Champ |Type |Description

|Nom
|Obligatoire
|Quel est le titre de cette fonctionnalité. Par exemple, "Skeletons"

|Rang d'affichage
|Facultatif
|Quel ordre il a en relation avec les autres squelettes. Avoir un nombre inférieur signifie un rang d'affichage plus élevé.
|===


.Ajouter un composant : Formulaire de squelette - Paramètres globaux
|===
|Champ |Type |Description

|Portées activées
|Facultatif
|Cochez cette case si vous voulez que le filtrage des portées dans ce composant.

|Secteur
|Facultatif
|Cette fonctionnalité appartient à [Objectif].

|Portée générale
|Facultatif
|

|Commentaires activés
|Facultatif
|Si vous voulez avoir des commentaires dans les rencontres.

|Longueur maximale des commentaires
|Facultatif
|Quelle est la limite de caractères que les participants ont pour faire des commentaires. Laisser 0 pour la valeur par défaut.

|Les autorisations d'actions peuvent être définies pour chaque squelette
|Facultatif
|Si vous voulez être en mesure de définir les permissions pour chaque squelette.

|Annonce
|Facultatif
|Une annonce générale qui sera visible sur la page d'accueil des squelettes généraux.
|===


.Ajouter un composant : Formulaire de squelettes - Paramètres de l'étape
|===
|Champ |Type |Description

|Commentaires désactivés
|Facultatif
|Si vous voulez activer les commentaires pour cette phase.

|Annonce
|Facultatif
|Une annonce générale qui sera visible sur la page d'atterrissage générale du squelette.
|===

.Nouveau formulaire de squelette
|===
|Champ |Type |Description

|Titre
|Obligatoire
|Quel est le titre de ce squelette. Par exemple, « District A »

|Rang d'affichage
|Facultatif
|Quel ordre aura-t-il en relation avec les autres squelettes. Avoir un nombre inférieur signifie un rang d'affichage plus élevé.
