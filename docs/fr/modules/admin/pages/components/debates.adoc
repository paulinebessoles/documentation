= Débats

== Understand the basics

Le volet Debate permet aux gens d'avoir des discussions, sans voter ni prioriser. Il permet également de clore les débats,
en ajoutant des conclusions ou un résumé de ce que la discussion portait.

image::components/debates/example01.png[Debates listing example]

== Configuration

=== Créer un composant de débat

Pour créer un composant Debate, allez d'abord dans xref:admin:spaces.adoc[espace participatif] où vous voulez le créer.

image::components/debates/component_debate.png[Add Debate component button]

Alors:

. Cliquez sur l'élément "Composants" dans le menu secondaire
. Cliquez sur le bouton "Ajouter un composant"
. Sélectionnez "Debate"
. Remplissez le formulaire de création de composant

Vous pouvez toujours mettre à jour ce formulaire et la configuration après avoir créé le composant, en cliquant sur l'image
image:icons/action_configure.png[Configurer l'icône] Configurer l'icône dans la liste des composants.

=== Gérer les débats

Pour commencer à créer et à gérer des débats, cliquez dans le titre du composant dans la barre de navigation ou dans la liste des composants, ou cliquez dans l'image
: icons/action_edit. Icône ng[Gérer l'icône du composant] dans la liste des composants.

image::components/debates/manage_debates.png[Manage debates table]

Sur la liste des débats dans le panneau d'administration, les administrateurs peuvent voir :

* *Titre* : Titre du débat.
* *Date de début* : Date de début du débat.
* *Date de fin*: Date de fin du débat.
* *Portée* : Portée du débat, si activé dans l'espace participatif.

Vous pouvez également exporter les commentaires dans les débats au format CSV, JSON et Excel en cliquant sur le bouton "Exporter tous".

=== Types de débats

Il est possible d'avoir deux types de débats :

. Finite : avec une date de début et de fin
. Ouvrir: sans date de début et de fin

Les participants peuvent voir les types de débats dans la liste publique du débat.

image:components/debates/debates_list.png[Liste des débats]

=== Fermer le formulaire de débat

Un débat peut être clos. Ceci est utile pour publier les conclusions des débats.
Après la fermeture d'un débat, il ne peut plus être édité ni commenté.

image::components/debates/close_debate_backend.png[Close debate backend form]

Prenez en compte que si le débat a été créé par un participant, alors seul ce participant peut le fermer.

image::components/debates/close_debate_frontend.png[Close debate frontend form]

.Fermer le formulaire de débat
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Conclusions
|Obligatoire
|Quelle est la conclusion de ce débat.
|===

Les conclusions sont montrées publiquement avec la date et l'heure limites.

image::components/debates/debate_closed.png[Closed debate]
