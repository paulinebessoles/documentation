= Réunions

== Understand the basics

Les réunions sont également les entités faisant partie du volet Réunions.
Ils sont généralement créés par les administrateurs, mais les participants peuvent également créer des réunions lorsque la fonctionnalité est activée.

Les administrateurs peuvent effectuer différentes actions lors de réunions, selon qu'elles sont créées à partir de l'administration ou par les participants.

== Réunions des participants

Les réunions peuvent être créées par les participants si le paramètre "Participants peuvent créer des réunions" est coché.
Si un participant appartient à un xref:admin:participants/groups.adoc[groupe d'utilisateurs], alors il est également en mesure de créer des réunions
comme ce groupe.

Lorsqu'un participant est sur la liste des réunions, il peut cliquer sur le bouton "Nouvelle réunion".

image::components/meetings/new_meeting_button_front.png[Create a new meeting button (frontend)]

Ensuite, ils sont redirigés sur le formulaire de création des réunions des participants.

== Réunions en ligne

Pour les réunions en ligne, selon le moment et la configuration de la réunion, différents messages sont affichés aux participants pour
les informer sur la façon de participer à la réunion en ligne.

Si l'URL de la réunion en ligne n'est pas encore définie, un message indiquant "Lien disponible bientôt" s'affiche.

image::components/meetings/online_meeting_available_soon.png[Online meeting available soon]

Si l'URL de la réunion en ligne est définie, quelques minutes avant le début de la réunion, le message dit « Le lien pour rejoindre
la réunion sera disponible quelques minutes avant le début »

image::components/meetings/online_meeting_available_before.png[Online meeting available before it starts]

Si l'URL de la réunion en ligne est définie, l'heure de début de la réunion est arrivée, et vous avez choisi le type d'intégration Iframe comme
"Ouvrir l'URL dans un nouvel onglet", puis le message dit que "la réunion se déroule en ce moment" avec un lien pour rejoindre la réunion.

image::components/meetings/online_meeting_happening.png[Online meeting available before it starts]

Si l'URL de la réunion en ligne est définie, l'heure de début de la réunion est arrivée, et vous avez choisi le type d'intégration Iframe comme
"Intégrer à la réunion", puis le message dit que "la réunion se déroule en ce moment" avec la réunion intégrée dans le bloc.

image::components/meetings/online_meeting_happening_embed.png[Online meeting available before it starts]
