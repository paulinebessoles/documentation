= Sondages

== Understand the basics

Meeting polls allow administrators to define questions for a meeting, that participants can answer
while it's happening. It can be very useful to gather quick feedback and make decisions during meetings without having to leave the meeting and use
other kind of tools.

Grâce à cette fonctionnalité, les administrateurs peuvent définir des questions et gérer lorsque ces questions et leurs résultats sont affichés
aux participants.

== Configuration

=== Create poll

Pour configurer les sondages pour une réunion spécifique, cliquez sur le bouton "Gérer les sondages": icons/action_list.png[Gérer le bouton des sondages] "Gérer les sondages".

Vous pouvez ajouter autant de questions et de réponses que vous le souhaitez.

=== Manage poll

Avant la réunion, si les administrateurs ont déjà configuré les questions des sondages, les participants peuvent voir un bouton "Répondre au sondage"
sur la page de la réunion, les avertissant qu'un sondage sera publié sur cette réunion.

image::components/meetings/polls/frontend_before_meeting.png[Reply poll page before meeting]

Lorsque la réunion commence, l'administrateur peut aller à la page d'administration du sondage dans le frontend pour commencer à envoyer les questions
aux participants.

image::components/meetings/polls/poll_administration_page.png[Polls administration page]

À leur fin, une fois qu'une question est envoyée, les participants la voient apparaître sur la page des sondages et peuvent y répondre.

image::components/meetings/polls/poll_participant_reply.png[Polls question for participant]

Ensuite, les administrateurs peuvent décider d'envoyer les résultats des questions aux participants, afin qu'ils sachent quelle option est la plus votée.

image::components/meetings/polls/poll_participant_replies_sent.png[Polls replies view for participant]

=== Exportation

Les réponses des participants peuvent être exportées via le bouton "Exporter tous" dans CSV, JSON, Excel et PDF.
