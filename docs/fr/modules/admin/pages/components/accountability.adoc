= Suivi des réalisations

== Understand the basics

Le volet responsabilisation permet aux participants de connaître le statut de la mise en œuvre d'un projet et de suivre son évolution.

This feature allows administrators to present a summary of the achievements decided upon in assemblies, during
participatory processes, or any other space. Les administrateurs créent des résultats qu'ils peuvent ensuite catégoriser et lier à des propositions, des réunions ou des projets d'investissement
(par exemple, dans le cadre d'un budget participatif).

image::components/accountability/example01.png[Example of results list]

Les résultats peuvent être surveillés de près, avec un historique de modifications, un pourcentage de réalisation et des jalons
indiquant leur progression. Les résultats, les jalons et les états peuvent être mis à jour manuellement via le panneau d'administration ou via un CSV.

image::components/accountability/example02.png[Example of a result]

== Configuration

=== Créer un composant de responsabilité

Pour créer un composant de responsabilisation, allez d'abord dans xref:admin:spaces.adoc[espace participatif] où vous voulez le créer.

image::components/accountability/component_accountability.png[Add accountability component button]

Alors:

. Cliquez sur l'élément "Composants" dans le menu secondaire
. Cliquez sur le bouton "Ajouter un composant"
. Sélectionnez "Accountability"
. Remplissez le formulaire de création de composant

Vous pouvez toujours mettre à jour ce formulaire et la configuration après avoir créé le composant, en cliquant sur l'image
image:icons/action_configure.png[Configurer l'icône] dans la liste des composants.

image::components/accountability/component.png[Add component form]

.Formulaire de responsabilisation
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Nom
|Obligatoire
|Quel est le titre de cette fonctionnalité. Par exemple "Responsabilité"

|Rang d'affichage
|Facultatif
|L'ordre du composant en relation avec les autres composants. Avoir un nombre inférieur signifie un rang d'affichage plus élevé.
Par exemple, si vous avez deux composants, "Composant a" avec le poids "10" et "Composant b" avec le poids "0"
puis "Composant b" est d'abord lors de la commande.
|===

.Paramètres généraux
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Portées activées
|Facultatif
|Cochez cette case si vous voulez que le filtrage xref:admin:settings/scopes.adoc[Scopes] soit disponible dans ce composant.

|Secteur
|Facultatif
|Ce composant appartient à xref:admin:settings/scopes.adoc[Scope].

|Commentaires activés
|Facultatif
|Si vous voulez avoir des commentaires dans le composant.

|Longueur maximale des commentaires
|Facultatif
|La limite de caractères que les participants ont lors de la rédaction de commentaires. Laisser 0 pour la valeur par défaut.

|Intro
|Facultatif
|Une introduction générale qui sera affichée sur la page d’atterrissage de la responsabilité.

|Afficher les progrès
|Facultatif
|Cochez cette case si vous voulez que la barre de progression soit affichée sur les résultats avec un niveau de progression aa.
|===

.Définition des étapes
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Commentaires désactivés
|Facultatif
|Si vous voulez activer les commentaires pour cette phase.
|===

=== Aperçu des résultats par Filtre de taxonomie.

Utilisez les filtres de taxonomie sur les résultats pour offrir aux participants un aperçu rapide de l’évolution du projet par taxonomie.

Pour cela :

1. Configurer une taxonomie avec certains sous-éléments et disponible pour votre espace actuel
1. Ajouter un filtre de taxonomie
1. Configurez les paramètres globaux du composant et ajoutez le filtre que vous avez créé
1. Voir la vue publique. Un bouton "Voir par: <Taxonomy name>" apparaîtra.

image::components/accountability/add_taxonomy_filter.png[Add taxonomy filter to your accountability component]

image::components/accountability/grouped_by_taxonomy.png[Participant can now group result by taxonomy]

=== Sous-résultats

Un résultat peut parfois être le produit d'autres résultats.
Par exemple, les résultats de la « construction de 3 patins en ville » utiliseraient trois sous-résultats pour surveiller une réalisation détaillée du résultat.

Créez un sous-résultat en cliquant sur le signe plus de la liste de responsabilités.
Pour accéder à la liste des sous-résultats, cliquez sur le titre principal du résultat pour y accéder.
