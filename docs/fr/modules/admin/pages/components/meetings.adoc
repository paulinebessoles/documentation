= Réunions

== En bref

Le volet Réunions permet aux administrateurs d'organiser, d'annoncer, de traiter des inscriptions et de clôturer des réunions en personne ou en ligne.
Selon la configuration du composant, les réunions ne peuvent être créées que par les administrateurs, ou aussi par les participants.

Dans cette section de documentation, vous pouvez trouver comment configurer et gérer tous ces éléments :

* xref:admin:admin:components/meetings/meetings.adoc[Meeting creation]: Créer des réunions officielles ou participantes.
* xref:admin:admin:components/meetings/registrations.adoc[Registrations]: Gérer comment les participants peuvent s'inscrire pour assister aux réunions.
* xref:admin:components/meetings/agenda.adoc[Agenda]: Annoncez les sujets des réunions à discuter.
* xref:admin:admin:components/meetings/polls.adoc[Polls]: Activer le vote rapide pendant une réunion en ligne.
* xref:admin:components/meetings/close.adoc[Close]: Ajoute le rapport final et les minutes de la réunion.

== Configuration

Pour créer un composant de réunions, allez d'abord sur xref:admin:spaces.adoc[espace participatif] où vous voulez le créer.

image::components/meetings/component_meeting.png[Add Meetings component button]

Ensuite :

. Cliquer sur l'élément "Fonctionnalités" dans le menu secondaire
. Cliquer sur "Ajouter une fonctionnalité"
. Sélectionnez "Réunions"
. Remplir le formulaire de création de fonctionnalité

Vous pouvez toujours mettre à jour ce formulaire et la configuration après avoir créé le composant, en cliquant sur l’icône
image:icons/action_configure.png[Configurer l’icône] dans la liste des composants.

image::components/meetings/component.png[Add a meetings component]

.Ajouter un composant - Formulaire de réunions
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Nom
|Obligatoire
|Quel est le nom de ce composant.

|Rang d'affichage
|Facultatif
|Position de la fonctionnalité par rapport aux autres fonctionnalités. Avoir un nombre inférieur signifie un rang d'affichage plus élevé.
Par exemple, si vous avez deux fonctionnalités, "Fonctionnalité a" de poids "10" et "Fonctionnalité b" de poids "0"
alors "Fonctionnalité b" est placée avant dans le listage.
|===

.Ajouter un composant - Formulaire de réunions: Paramètres globaux
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Portées activées
|Facultatif
|Cochez cette case si vous voulez que le filtrage des portées dans ce composant.

|Secteur
|Facultatif
|Ce composant appartient à xref:admin:settings/scopes.adoc[Scopes].

|Annonce
|Facultatif
|Une annonce générale visible dans la liste des réunions dans l'interface participante.

|Conditions d'inscription par défaut
|Facultatif
|Les conditions de service que le participant doit accepter lors de son inscription.

|Commentaires activés
|Facultatif
|Si vous voulez que la création de commentaires soit activée dans les réunions.

|Longueur maximale des commentaires
|Facultatif
|La limite de caractères que les participants ont lors de la rédaction de commentaires. Laisser 0 pour la valeur par défaut.

|Code d'enregistrement activé
|Facultatif
|Cochez cette case si vous souhaitez donner automatiquement des codes d'inscription aux participants.
En savoir plus sur xref:components/meetings/registrations.adoc[Registrations].

|Les autorisations d'actions peuvent être définies pour chaque réunion
|Facultatif
|Si vous voulez des permissions plus granulaires à l'intérieur de chaque réunion.

|Activer la création de pads
|Facultatif
|Si vous voulez un Etherpad intégré à vos réunions. Doit avoir l'intégration xref:services:etherpad.adoc[Etherpad] configurée.

|Les participants peuvent créer des réunions
|Facultatif
|Permet aux participants de créer de nouvelles réunions. En savoir plus dans xref:components/meetings/meetings.adoc#_participants_meetings[Participants meetings].

|Cartes activées
|Facultatif
|Si vous voulez afficher une carte avec les adresses des réunions dans la liste et les pages des réunions.
|===

.Ajouter un composant - Formulaire de réunions: Paramètres de l'étape
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Annonce
|Facultatif
|Une annonce générale visible dans la liste des assemblées générales pour cette phase.

|Commentaires activés
|Facultatif
|Si vous voulez que la création de commentaires soit activée dans les réunions pour cette phase.
|===
