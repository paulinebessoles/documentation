= Évolution du projet

== En bref

Pour que les participants puissent suivre et mener l'évolution des projets, les administrateurs peuvent créer une chronologie
en cliquant sur l'[Icône d'évolution du projet].

image::components/accountability/results_entries.png[Result page with timeline entries]

== Configuration

image::components/accountability/timeline_list.png[Timeline entries list]

Ici, il est possible de créer et gérer les différentes phases de mise en oeuvre de la réalisation, et ajouter de nouvelles données dans la chronologie.

image::components/accountability/new_entry.png[New timeline entry form]


.Tableau de données chronologiques
|===
|Champ |Type |Description

|Date
|Obligatoire
|Format attendu : jj/mm/aaaa

|Titre
|Obligatoire
|Titre de la donnée chronologique.

|Description
|Facultatif
|Description de la donnée chronologique.
|===

Les données chronologiques sont ensuite disponible sur la page des réalisations.
