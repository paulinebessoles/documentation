= Gestion de la responsabilité

== Understand the basics

Les résultats sont les entités qui constituent la composante de la responsabilité. They can be linked to proposals, projects, meetings, and have
statuses, and dates assigned.

Les administrateurs peuvent également créer des sous-résultats, pour détailler les éléments nécessaires à une implémentation de résultats.

== Créer un résultat

Pour commencer à créer des résultats, cliquez sur le titre du composant dans la barre de navigation ou dans la liste des composants, ou cliquez sur l'image
: icônes/action_edit. Icône ng[Gérer l'icône du composant] dans la liste des composants.

image::components/accountability/new_result_button.png[New result button]

Ensuite, cliquez sur le bouton "Nouveau résultat" en haut à droite de la page.

== Gérer les résultats

Sur la liste des résultats dans le panneau d'administration, les administrateurs peuvent voir :

* *ID* : Numéro d'identification pour ce résultat.
* *Titre* : Titre du résultat.
* *Catégorie* : Catégorie du résultat.
* *Portée* : Portée du résultat, si activé dans l'espace participatif.
* *xref:admin:admin:components/Accounability/statuses.adoc[Status]*: Statut du résultat.
* *Progression* : Pourcentage de progression.
* *Créé*: Date de création du résultat.

Vous pouvez filtrer et trier cette liste en cliquant sur le bouton "Filtre" et sur les en-têtes des colonnes.
Vous pouvez également rechercher des résultats par leur ID et leur titre dans la barre de recherche.
