= Progrès

== En bref

Le progrès représente l'état d'une réalisation en cours.
Pour les participants, les progrès indiquent clairement l'état de chaque réalisation, car les progrès sont toujours affichés avec les réalisations.
Pour les administrateurs, les progrès aident à garder à jour l'avancement des réalisations, puisque chaque progrès peut avoir un pourcentage d'avancement par défaut.

Après la configuration des progrès, il est possible d'assigner [un progrès à une réalisation].
Une fois le progrès d'une réalisation mis à jour, son avancement est automatiquement défini par défaut sur l'état défini dans le progrès.

La mise à jour du progrès d'une réalisation ne mettra pas à jour l'historique de la réalisation, mais peut [envoyer des notifications].

== Configuration

Pour créer des Progrès, cliquez sur "Progrès" en haut à droite de la liste des réalisations.

image::components/accountability/statuses_button.png[Statuses button]

Ici, il est possible de créer et gérer les différents progrès pour les réalisations.

image::components/accountability/statuses_list.png[Statuses list]

Cliquer "Nouveau statut" pour commencer à créer des progrès.

[Nouveau formulaire de progrès]


.Formulaire de nouveau statut
|===
|Champ |Type |Description

|Clé
|Obligatoire
|Utilisé comme identifiant unique pour [import CSV].

|Nom
|Obligatoire
|Nom pour du progrès.

|Description
|Facultatif
|Description pour du progrès.

|Avancement
|Facultatif
|Avancement par défaut pour un progrès assigné à cet état (pourcentage de 0 à 100).
