= Pages

== En bref

La fonctionnalité Page permet aux participant·e·x·s d'avoir plus d'information sur un espace. Les administrateur·ices peuvent remplir une page avec des informations, des images, des liens, afin d'informer et mettre à jour les participant·e·x·s.

Similaire à xref:admin:pages.adoc[Pages], la principale différence étant que cette fonctionnalité permet de définir des pages à l'intérieur de
un espace participatif, et non dans la section Aide.

image::components/page/example01.png[Example of a page component in a participatory process]

== Configuration

=== Créer une fonctionnalité de page

Pour créer une fonctionnalité Page, aller d'abord dans xref:admin:spaces.adoc[espace participatif] à l'endroit de création souhaité.

image::components/page/component_page.png[Add Page component button]

Ensuite :

. Cliquer sur l'élément "Fonctionnalités" dans le menu secondaire
. Cliquer sur "Ajouter une fonctionnalité"
. Sélectionner "Page"
. Remplir le formulaire de création de fonctionnalité

Il est toujours possible de mettre à jour ce formulaire et la configuration après avoir créé la fonctionnalité, en cliquant sur l’icône
image:icons/action_configure.png[Configurer l’icône] dans la liste des fonctionnalités.

image::components/page/component.png[Add component form]

.Ajouter un composant : Formulaire de page
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Nom
|Obligatoire
|Le titre de cette fonctionnalité.

|Rang d'affichage
|Facultatif
|Position de la fonctionnalité par rapport aux autres fonctionnalités. Avoir un nombre inférieur signifie un rang d'affichage plus élevé.
Par exemple, si vous avez deux fonctionnalités, "Fonctionnalité a" de poids "10" et "Fonctionnalité b" de poids "0"
alors "Fonctionnalité b" est placée avant dans le listage.
|===

.Ajouter une fonctionnalité : Formulaire de page - Paramètres généraux
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Annonce
|Facultatif
|Une annonce générale visible en haut de la page.
|===

.Ajouter une fonctionnalité : Formulaire de pages - Paramètres de l'étape
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Annonce
|Facultatif
|Une annonce générale visible en haut de la page pour cette phase.
|===

=== Créer une page

Une fois la fonctionnalité de la page créée, cliquer sur l'icône Modifier pour rédiger le contenu à afficher sur la page.

image:components/page/new_page.png[Formulaire de nouvelle page]

.Formulaire de nouvelle page
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Corps
|Obligatoire
|Le contenu affiché sur la page. Il est possible d'ajouter des images, des liens, des vidéos, du formatage, etc.
|===
