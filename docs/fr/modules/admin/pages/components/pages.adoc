= Pages

== Understand the basics

La fonctionnalité Page permet aux participants d'avoir plus d'information sur un espace. Les administrateurs peuvent remplir une page avec du contenu d'information, des images
, des liens, pour informer et mettre à jour les participants.

C'est la même chose que les pages générales xref:admin:pages. doc[Pages], la principale différence étant que ce composant permet de définir des pages à l'intérieur de
un espace participatif, et non dans la section Aide.

image::components/page/example01.png[Example of a page component in a participatory process]

== Configuration

=== Créer un composant de page

Pour créer un composant de page, allez d'abord dans xref:admin:spaces.adoc[espace participatif] où vous voulez le créer.

image::components/page/component_page.png[Add Page component button]

Alors:

. Cliquez sur l'élément "Composants" dans le menu secondaire
. Cliquez sur le bouton "Ajouter un composant"
. Sélectionnez "page"
. Remplissez le formulaire de création de composant

Vous pouvez toujours mettre à jour ce formulaire et la configuration après avoir créé le composant, en cliquant sur l’icône
image:icons/action_configure.png[Configurer l’icône] dans la liste des composants.

image::components/page/component.png[Add component form]

.Ajouter un composant : Formulaire de page
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Nom
|Obligatoire
|Le titre de ce composant.

|Rang d'affichage
|Facultatif
|L'ordre du composant en relation avec les autres composants. Avoir un nombre inférieur signifie un rang d'affichage plus élevé.
Par exemple, si vous avez deux composants, "Composant a" avec le poids "10" et "Composant b" avec le poids "0"
puis "Composant b" est d'abord lors de la commande.
|===

.Ajouter un composant : Formulaire de page - Paramètres globaux
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Annonce
|Facultatif
|Une annonce générale visible en haut de la page.
|===

.Ajouter un composant : Formulaire de pages - Paramètres de l'étape
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Annonce
|Facultatif
|Une annonce générale visible en haut de la page pour cette phase.
|===

=== Créer une page

Une fois que vous avez créé le composant de la page, cliquez sur l'icône Modifier pour écrire le contenu à afficher sur la page.

image:components/page/new_page.png[Formulaire de nouvelle page]

.Formulaire de nouvelle page
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Corps
|Obligatoire
|Le contenu affiché sur la page. Vous pouvez ajouter des images, des liens, des vidéos, du formatage, etc.
|===
