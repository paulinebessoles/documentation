= Options spécifiques

== En bref

La fonctionnalité Propositions a été construite avec des mises en page différentes, afin de traiter différents cas d'utilisation.
Ces configurations spécifiques sont plus rarement utilisées et nécessitent une bonne connaissance du logiciel.

== Configuration

Les trois fonctionnalités peuvent être activées dans les paramètres globaux du formulaire de configuration de la fonctionnalité de proposition.

image::components/proposals/component_specific_conf.png[Add component form - Specific configurations]

Dans cette section, vous pouvez apprendre à configurer et à utiliser les fonctionnalités suivantes :
