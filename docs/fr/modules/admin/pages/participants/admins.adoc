= Administrateurs

== En bref

Les administrateur·ices sont les personnes qui gèrent l'organisation, créent des processus, et gère la plateforme dans son ensemble
Il y a deux types d'administrateur·ices :

* Administrateur·ices : ont accès à toute la configuration de la plateforme.
* Gestionnaires de participant·e·x·s : seul·e·x·s les participant·e·x·s xref:admin:participants/impersonations.adoc[impersonations] peuvent être géré·e·x·s.

image:participants/admins.png[Liste des admins]

== Configuration

=== Inviter unx administrateur·ice

Pour inviter unx administrateur,·ice cliquer sur le bouton "Nouvel·le·x administrateur·ice" en haut à droite de la page.
Vous pouvez inviter unx participant·e·x déjà inscri·e·x ou non enregistré·e·x.

=== Gérer les administrateurs

Vous pouvez trouver touxtes les administrateur·ices dans la liste des administrateur·ices dans le panneau d'administration, avec :

* *Rôle* : le type de droits d'administrateur·ice, ne peut être que Administrateur·ice ou Gestionnaire de Participant·e·x·s.
* *Nom*: nom de lae participant·e·x.
* *E-mail*: email de lae participant·e·x.
* *Invitation envoyée à* : date et heure de l'invitation par e-mail.
* *Invitation acceptée à* : quand iel a accepté l'invitation par mail.
* *Date de dernière connexion*: date de la dernière connexion de l'administrateur·ice.
* *Date de création* : date de création du compte participant·e·x.

Il est possible de filtrer et trier cette liste en cliquant sur "Filtrer" et sur les en-têtes des colonnes.
Vous pouvez également rechercher des administrateur·ices par e-mail, nom ou surnom dans la barre de recherche.

=== Révoquer les autorisations d'administrateur·ice

Lorsqu'unx administrateur·ice n'a plus besoin d'accès, vous devriez révoquer ses droits en cliquant sur le bouton "Supprimer".
Vous devez confirmer la suppression des droits pour supprimer complètement leurs droits.
La révocation des autorisations d'administrateur·ice ne supprime pas le compte de lae participant·e·x·s.
