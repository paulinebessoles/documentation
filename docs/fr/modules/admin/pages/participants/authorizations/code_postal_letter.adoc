= Code par lettre postale d'autorisation

== En bref

Cette autorisation permet à unx participant·e·x de demander un code de vérification à envoyer à une adresse, pour confirmer qu'iels
vivent ou travaillent à cet endroit. Iels doivent ensuite remplir le code reçu dans la plateforme.

NOTE: Si vous ne voulez pas collecter de données, vous devez le désactiver dans le panneau système xref:configure:system.adoc[System panel].

== Configuration

Le processus est le suivant :

. Lea participant·e·x demande un code de vérification à envoyer à une adresse.
. Unx administrateur·ice envoie la lettre à son adresse avec le code de vérification.
. Unx administrateur·ice marque la lettre comme envoyée.
. Une fois la lettre marquée comme envoyée, lea participant·e·x est en mesure d'introduire le code.
. Si ce code correspond, alors il sera validé.

=== 1. Demande de code de vérification

En se rendant dans la section de l'autorisation sur son profil, ou en cliquant sur le bouton Action, si cette autoristation est requise, unx participant·e·x pourra voir le formulaire demandant son adresse complète.

image::participants/authorizations_code_postal_letter_request.png[Code by postal letter request by participant]

En modifiant le formulaire, le message suivant s'affiche :

image::participants/authorizations_code_postal_letter_edit.png[Edit the postal letter request]

Ce message s'affiche si la lettre est en cours d'envoi.  Si cela doit être corrigé, alors le participant a besoin de
pour contacter un administrateur de la plateforme.Pour modifier, lea participant·e·x devra contacter unx administrateur·rice de la plateforme.

=== 2. Vérification par l'administateur·ice

Dans le panneau d'administration, l'administrateur·ice peut consulter la demande en allant à la section « Code par lettre postale » de la section « Participant·e·x·s ».

image::participants/authorizations_code_postal_letter_ongoing.png[Code by postal letter ongoing panel in administration]

Ici, il est possible de voir le code de vérification et l'envoyer aux participant·e·x·s par lettre postale. Une fois le code envoyé, iels peuvent le marquer
comme envoyé l'icône image:icons/action_verify.png[Marquer comme envoyé]. Il sera également affiché quand la lettre a été envoyée.

=== 3. Valider le code de lae participant·e·x

Une fois le code parvenu à lae participant·e·x, iel peut aller dans les paramètres de son compte et cliquer dans la méthode de vérification "Code par lettre postale"
.

image::participants/authorizations_code_postal_letter_verify.png[Code by postal letter ongoing in the participant account]

Ensuite, iels peuvent remplir le formulaire avec le code reçu.

image::participants/authorizations_code_postal_letter_confirm.png[Code by postal letter confirmation form]

=== 4. Correspondance de code

Si le code de vérification correspond, alors lae participant·e·x voit le message "Félicitations. Vous avez été vérifié·e·x avec succès."

Si le code de vérification ne correspond pas, alors lae participant·e·x voit un message d'erreur demandant de vérifier à nouveau le code dans
la lettre.

Une fois la vérification complétée, elle est visible dans les paramètres du compte de lae participant·e·x, comme pour les autres vérifications.

image::participants/authorizations_code_postal_letter_verified.png[Verified by code by postal letter]
