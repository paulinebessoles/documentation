= Code par lettre postale d'autorisation

Cette autorisation permet à un participant de demander un code de vérification à envoyer à une adresse, pour confirmer qu'ils
vivent ou travaillent dans cet endroit. Ils doivent ensuite remplir le code reçu dans la plate-forme.

NOTE: Si vous ne voulez pas collecter de données, vous devez le désactiver dans le panneau système xref:configure:system.adoc[System panel].

Le processus est :

. Le participant demande un code de vérification à envoyer à une adresse.
. Un administrateur envoie la lettre à son adresse avec le code de vérification.
. Un administrateur marque la lettre comme envoyée.
. Une fois que vous avez marqué la lettre comme envoyée, le participant est en mesure d'introduire le code.
. Si ce code correspond, alors il sera vérifié.

== 1. Demande de code de vérification

By going to the authorization section in its profile, or by clicking on the action button if this permission is requested,
a participant can see the form requesting their full address.

image::participants/authorizations_code_postal_letter_request.png[Code by postal letter request by participant]

S'ils essaient de le modifier, ils voient ce message :

image::participants/authorizations_code_postal_letter_edit.png[Edit the postal letter request]

C'est parce que la lettre pourrait être en cours d'envoi. Si cela doit être corrigé, alors le participant a besoin de
pour contacter un administrateur de la plateforme.

== 2. Revue de l'administrateur

Dans le panneau d'administration, l'administrateur peut consulter la demande en allant à la section « Code par lettre postale » de la section « Participants ».

image::participants/authorizations_code_postal_letter_ongoing.png[Code by postal letter ongoing panel in administration]

Ici, ils peuvent voir le code de vérification et l'envoyer au participant par lettre postale. Une fois le code envoyé, ils peuvent le marquer
en tant que tel avec l'icône image:icons/action_verify.png[Marquer comme envoyé]. Il sera également affiché quand la lettre a été envoyée.

== 3. Valider le code du participant

Une fois que le participant a reçu le code, il peut aller dans les paramètres de son compte et cliquer dans la méthode de vérification "Code par lettre postale"
.

image::participants/authorizations_code_postal_letter_verify.png[Code by postal letter ongoing in the participant account]

Ensuite, ils peuvent remplir le formulaire avec le code qu'ils ont reçu.

image::participants/authorizations_code_postal_letter_confirm.png[Code by postal letter confirmation form]

== 4. Correspondance de code

Si le code de vérification correspond, alors le participant voit le message "Félicitations. Vous avez été vérifié avec succès. »

D'un autre côté, si le code de vérification ne correspond pas, alors le participant voit une erreur demandant de vérifier le code dans
la lettre.

Une fois qu'un participant est vérifié, comme avec d'autres vérifications, il peut le voir dans les paramètres de son compte.

image::participants/authorizations_code_postal_letter_verified.png[Verified by code by postal letter]
