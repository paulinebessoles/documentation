= Autorisation de recensement de l'organisation

== En bref

L'Autorisation de recensement de l'organisation permet aux administrateurices de télécharger une liste de mails de paprticipant·e·x·s, pour que cette liste puisse effectuer des actions spécifiques, telles la création d'une proposition dans un espace spécific, etc.

== Configuration

Le processus est le suivant:

. Les administrateur·ices téléchargent un CSV avec les mails vérifés de participant·e·x·s
. Les participant·e·x·s font une demande de vérification sur leur compte
. Seul·e·x·s les participant·e·x·s avec un email dans ce fichier CSV peuvent être vérifiés

NOTE: Cette autorisation est trop basique et vous avez besoin de plus de fonctionnalités liées aux importations de recensement CSV?
Vérifier tous les CSV ou fichiers d'autorisations dans la page https://decidim.org/modules/[Modules].

=== 1. Téléchargement CSV

Dans le panneau d'administration, vous pouvez télécharger un fichier CSV en cliquant sur "Recensement de l'organisation" dans la section "Participant·e·x·s".

image::participants/authorizations_organization_census.png[Organization's census in administrator panel]

Une fois que le CSV est importé, l'administrateur·ice peut voir combien de participant·e·x·s ont été importé·e·x·s. Iels peuvent également supprimer ce recensement
importé.

image::participants/authorizations_organization_census_imported.png[Imported CSV]

=== 2. Demande de participation

En se rendant dans la section des autorisations dans son profil, ou en cliquant sur le bouton d'action, si cette autorisation est requise, unx participant·e·x peut demander à être vérifié·e·x.

image::participants/authorizations_organization_census_account.png[Organization's census in participant's accounts]

=== 3. Vérification des participant·e·x·s

Après vérification, comme avec d'autres vérifications, elles sont visibles dans les paramètres de leur compte.

image::participants/authorizations_organization_census_verified.png[Verified by organization's census]
