= Premiers pas dans Decidim

== En bref

=== Connectez-vous pour la première fois

Une fois Decidim installé, par vous ou votre developer, et qu'une organisation est créée dans le panneau Système, vous devriez recevoir un email vous demandant de compléter la création de votre compte.
Si vous n'avez pas reçu d'email d'invitation, demandez à la personne responsable de l'administration du système de vous ajouter en tant qu'administrateur·ice.

image::first_steps_invitation_email.png[Administrator invitation email]

L'email d'invitation vous redirige vers un formulaire pour que vous puissiez terminer la configuration de votre compte avec votre surnom, votre nom et votre mot de passe.

image::first_steps_account_creation.png[Administrator invitation email]

=== Accéder au tableau de bord de l'administration

Une fois conné·e·x, vous pouvez voir la plateforme en tant que participant·e·x, mais vous ne pouvez pas configurer et gérer la plateforme.
Pour accéder au tableau de bord de l'administration, ou le panneau d'administration, il est possible de cliquer sur trois éléments différents:

. Le bouton "Tableau de bord de l'administration" à gauche de l''en-tête de l'administration, que seul·e·x·s les administrateur·e·x·s peuvent voir.
. Le bouton "Modifier" à droit de l'en-tête de l'administration.
. Le bouton "Tableau de bord de l'admin" dans la liste déroulante qui apparait en cliquant sur l'avatar de votre compte.

image::admin_dashboard_menu.png[Decidim Admin Dashboard in user menu]

Le bouton "Modifier" redirige vers l'administration de l'espace visible dans le panneau participant·e·x, alors que les deux autres boutons redirigent vers le tableau de bord de l'administration.

=== Condition de service des administrateur·ices

En accédant pour la première fois l'administration de Decidim, il vous est demandé·e·x de consulter les conditions de service de l'administration.
Cliquer sur "J'accepte les conditions" pour pouvoir débuter la configuration de la plateforme.

image::admin_tos_pending.png[Decidim Admin Dashboard in user menu]

Ces conditions de service peuvent vous être utiles si vous commencez à inviter d'autres administrateur·ices sur la plateforme et que vous souhaitez les informer des règles spécifiques en matière de confidentialité ou de tout autre élément pertinent avant qu'iles n'aient accès à l'administration.

Ces conditions de service peuvent être configurées dans le xref:admin:settings/configuration.adoc[Panneau de configuration].

=== Explication du tableau de bord

Une fois les conditions de service des administrateur·ices acceptées, vous pouvez voir le tableau de bord.
Sur une instance nouvellement créée, presque aucune donnée n'est visible, mais il est possible de voir à quoi ressemble une instance active :

image::admin_dashboard.png[Decidim Admin Dashboard]

Sur cette page, il vous est possible de voir plusieurs sections :

. *Modérations en attente" : met en évidence le nombre de modérations en attente, les utilisateur·ices signalé·e·x·s et les contenus combinés.
. *Activité" : nombre de participant·e·x·s et adminstrateur·ices s'étant connecté·e·x·s le mois précédent, la semaine précédente et la veille.
. *Métriques* : les graphiques d'évolution des différentes fonctionnalités dans Decidim.
. *Journal Admin* : les dernières actions effectuées par touxtes les administrateur·ices sur la plateforme.
