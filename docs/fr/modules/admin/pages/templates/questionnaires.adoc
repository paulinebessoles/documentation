= Modèles de questionnaires

== En bref

Dans cette section, vous pouvez créer, modifier et supprimer des modèles de questionnaires, que vous pouvez ensuite utiliser dans n'importe quelle fonctionnalité d'enquête.

== Configuration

=== Créer un modèle de questionnaire

Pour créer un nouveau modèle de questionnaire, vous devez cliquer sur le bouton "Nouveau modèle".

image::templates/templates_questionnaires_backend_form.png[Backend: new template form]

.Formulaire Nouveau modèle de questionnaire
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Nom
|Obligatoire
|Titre de ce modèle. Choisissez un titre clair, décrivant précisément à quoi sert l'enquête, comme « enquête de rétroaction ».

|Description
|Obligatoire
|Description des raisons de ce modèle. Cette description n'est disponible que dans le paneau des modèles, pour décrire les raisons précises de l'enquête.
|===

Cliquez sur le bouton "Enregistrer" pour continuer à créer votre questionnaire de modèle.

image::templates/templates_questionnaires_backend_survey_form.png[Backend: new survey template form]

Ensuite, cliquez sur le bouton "Modifier". Vous entrez le processus de création de l'enquête. Cliquez sur le bouton « Passer » si vous ne voulez pas utiliser un modèle existant.

Sur la page suivante, vous pouvez créer un sondage. Veuillez vous référer à la documentation xref:admin:components/surveys.adoc[Enquêtes] pour plus d'informations.

Cliquez sur le bouton "Enregistrer" pour voir un aperçu de votre enquête.

image::templates/templates_questionnaires_backend_survey_preview.png[Backend: new survey template preview]

Cliquez à nouveau sur le bouton "Enregistrer" pour enregistrer votre modèle.

=== Gérer les modèles

Tous les questionnaires apparaissent dans la liste des questionnaires :

* *Modèle*: nom du modèle.
* *Titre du questionnaire*: nom de l'enquête par défaut lors du choix de ce modèle.
* *Nombre de questions* : nombre de questions différentes dans l'enquête.
* *Créé le* : date de création.

Vous pouvez modifier un modèle et ses questions à tout moment. Veuillez également noter que lorsque vous choisissez un modèle lors de la création d'une enquête,
vous pouvez également modifier les questions, en ajouter ou en supprimer pour cette fonctionnalité spécifique.

=== Utiliser le modèle

Pour utiliser le modèle, créez une fonctionnalité xref:admin:components/surveys.adoc[Enquête] dans un espace participatif, puis modifiez-le.
Vous voyez ensuite une page vous proposant de choisir un modèle pour ce sondage.
