= Bloquer les messages d'utilisateur·ices

== En bref

Dans cette section, vous pouvez créer, modifier, supprimer des modèles bloquer les utilisateur·ice, que vous pouvez ensuite utiliser lors de la modération des utilisateur·ices.

== Configuration

=== Créer un modèle bloquer les messages d'utilisateur·ice

Pour créer un nouveau modèle pour bloquer des utilisateur·ices, vous devez cliquer sur "Nouveau modèle".

image::templates/templates_block_user_backend_form.png[Backend: new template form]

.Formulaire Nouveau modèle de questionnaire
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Nom
|Obligatoire
|Titre de ce modèle. Choisissez un titre clair, décrivant précisément à quoi sert le message, comme "spammer les utilisateurs".

|Description
|Obligatoire
|Bloquer le message utilisateur·ice qui est utilisé pour justifier le blocage de lae participant·e·x.
|===

=== Gérer

Tous les modèles Bloquer les messages d'utilisateur·ices apparaissent dans la liste des modèles :

* *Modèle*: nom du modèle.
* *Créé le* : date de création.

Vous pouvez modifier un modèle à tout moment.

=== Utiliser le modèle

Pour utiliser le modèle, allez dans le panneau xref:admin:moderations/reported_users.adoc[Utilisateur·ices signalé·e·x·s] et cliquez sur l'icône "Bloquer l'utilisateur·ice".
Vous voyez ensuite une page vous proposant de choisir un modèle de justification pour bloquer lae participant·e·x.
