= Bloquer les modèles de messages utilisateur

== En bref

Dans cette section, vous pouvez créer, modifier et supprimer des modèles de justification d'utilisateur de bloc, que vous pouvez ensuite utiliser lors de la modération des utilisateurs.

== Configuration

=== Créer un modèle de message utilisateur de bloc

Pour créer un nouveau modèle de message utilisateur de blocage, vous devez cliquer sur le bouton "Nouveau modèle".

image::templates/templates_block_user_backend_form.png[Backend: new template form]

.Nouveau formulaire de questionnaire
[cols="20h,10h,~"]
|===
|Champ |Type |Description

|Nom
|Obligatoire
|Titre de ce modèle. Choisissez un titre clair, décrivant précisément à quoi sert le message, comme "spammer les utilisateurs".

|Description
|Obligatoire
|Bloquer le message utilisateur qui est utilisé pour justifier le blocage du participant.
|===

=== Gérer

Tous les modèles de messages utilisateur bloqués apparaissent dans la liste des gabarits :

* *Modèle*: nom du modèle.
* *Créé à* : date de création.

Vous pouvez modifier un modèle à tout moment.

=== Actions

.Actions
[cols="10h,20h,~"]
|===
|Icône |Nom |Définition

|image:icons/action_edit.png[Modifier le modèle]
|Editer
|Éditer le message de l'utilisateur de blocage.

|image:icons/action_duplicate.png[Dupliquer le modèle]
|Duplicate
|Pour dupliquer le modèle.

|image:icons/action_delete.png[Supprimer le modèle]
|Supprimez
|Supprimer le modèle.
|===

=== Utiliser le modèle

Pour utiliser le modèle, allez dans le panneau xref:admin:moderations/reported_users.adoc[reporteds] et cliquez sur l'icône "Bloquer l'utilisateur".
Vous voyez ensuite une page vous proposant de choisir un modèle de justification pour bloquer ce participant.
