= Sections d'aide

== Understand the basics

Sur chaque index d'espace participatif et page de destination, vous pouvez voir un bouton "Aide" sur la bordure droite de la page.

image::settings/help_sections_frontend_hide.png[Help sections: hide]

En cliquant dessus, un visiteur ou un participant peut voir un message pop-up pour comprendre ce qu'est l'espace participatif.

image::settings/help_sections_frontend.png[Help sections: frontend]

Les administrateurs peuvent personnaliser ces *sections d'aide* pour les adapter à la spécificité de l'instance.

image::settings/help_sections_backend.png[Help sections: backend]

== Configuration

=== Désactiver

Vous pouvez masquer ces boîtes d'information en laissant tout le contenu vide pour chaque langue.

image::settings/help_sections_disable_backend.png[Disable help section in administration]
