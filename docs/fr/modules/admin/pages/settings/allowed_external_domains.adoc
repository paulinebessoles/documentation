= Domaines externes autorisés

== En bref

Pour se protéger contre les spams, et augmenter la sécurité sur votre instance, Decidim peut identifier tous les liens qui dirigent
vers une page Web externe, et ajouter le pop-up d'alerte suivant après le clic de l'utilisateur·ice :

image::settings/allowed_external_domains_pop-up.png[Pop-up that appears when clicking any external domains on Decidim]

Les participant·e·x·s sont averti·e·x·s qu'iels quittent votre instance.

== Configuration

=== Autoriser un domaine externe

Il peut y avoir des redirections pour lesquelles vous ne voulez pas que cette fonctionnalité soit activée.
Par exemple, si votre site est déployé à une URL comme `+decidim.example.org+` et que vous voulez que chaque URL `+example.org+` contourne le message d'alerte, il est possible d'ajouter votre URL, et d'en ajouter autant que souhaité, dans la page Domaines externes autorisés.

Pour ce faire, cliquez sur le bouton "Ajouter à la liste autorisée" et entrez l'URL du site Web que vous voulez autoriser.

image::settings/allowed_external_domains_admin_panel.png[Allowed external domains panel]

En cliquant sur un URL de ces domaines, les participant·e·x·s ne verront pas le message d'alerte.
